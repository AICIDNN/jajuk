Windows Installation Guide

Targeted platforms

This documentation targets: Windows 98, ME, 2000, XP, 2003, Vista.

Requirements

Java Runtime environment (JRE) 1.5 and above (1.6 recommended). You can get it for free at http://www.java.com.

Installation

We provide a self-extracting Windows archive (Jajuk-${version}.exe). Double click on it and follow installation.

Run

Launch Jajuk by clicking on Jajuk icon : Start / Jajuk / Jajuk

Upgrade

If you already have an installed Jajuk release:

  • It is safe to install the new release over previous one without uninstalling, Jajuk supports backwards compatibility. Your collection and configuration is kept and is stored under {your home directory}/.jajuk

  • Jajuk doesn't support downwards compatibility: A Jajuk release may not work with a collection built by a newer release than the one you use.

Uninstallation

  • Launch uninstaller by clicking on 'Uninstall' icon next to 'Jajuk ' one

  • Remove {user home}/.jajuk directory for each user you want to uninstall.
