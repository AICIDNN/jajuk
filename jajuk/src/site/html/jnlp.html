Launch jajuk online

Click on "Web Start me now" button to launch full Jajuk release
Java Web Start auto-detection :

Notes:

  • Jajuk is self-signed so you will be prompt to accept Jajuk launching: accept safely temporary or permanently, it is exactly the same code than the standalone installer.

  • Application works offline.

  • You need JRE 1.5.0 and above and JRE 1.5.0_6 and above under Linux (auto-installed under Windows/IE).

  • [Linux only] Automatic shortcut and menu creation doesn't work, install RPM instead or create shortcuts manually with following command: javaws http://jajuk.info:8001/jajuk-notest/app or launch javaws, select jajuk and execute.
