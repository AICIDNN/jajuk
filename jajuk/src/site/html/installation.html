For all others OS supporting Java 1.5 (Solaris, AIX, FreeBSD...) :

Please use the Java Installer .. This distribution file is a IzPack setup file working with any operating system.

  1. Launch it by typing in a console:  java -jar jajuk-java-installer-${version}.jar
  2. Select your language for installation
  3. Read carefully the provided README file
  4. Read and accept GPL license.
  5. Select installation target : Example: /usr/local/jajuk under Unix
  6. Create shortcuts (if your OS is support it)
