Jajuk 1.3 - New and Noteworthy

Jajuk team is pleased to present this new major upgrade that intend to reply to users most urgent requests: new audio formats support, sound breaks and memory footprint reduction. This release brings as well major GUI improvements and many new features.

Sound engine

New Audio formats supported Support for Flac, WMA, Real Player Audio has been added (requires mplayer 1.0pre8 or later under Linux and Mac). See current formats support list.

Better support for current formats o Better sound quality
o Seeking inside tracks is much faster
o No more sound skippings
o Ogg seeking now available

Full queue now remembered at next startup
 When using Start last track startup option, full queue is now remembered at next Jajuk session and not only last track

Interface

New visual identity Jajuk project has now an official graphist that designed new icons, splashscreens and website

New perspective selector Perspective selector is bigger and adopted Firefox look and feel

Hidable views 
 Hardly used views can now be hided for space saving (it is the new default Playlists Repository view)
Dynamic toolbars Toolbars are now dynamic and can be moved anywhere in the entire Jajuk frame (at the left / right / top or bottom). Their new positions are saved between jajuk sessions.

Hotkeys and Intellitype keyboard support (Windows only)
  • Jajuk keystrokes now works from the entire desktop and not only when Jajuk get focus (hotkey)
  • Jajuk main media functions now works from Microsoft Intellitype keyboads

New features

Reporting Jajuk now provides a collection reporter giving the ability to produce an HTML or XML collection report. It can be applied on the entire collection or from lower levels (artists, device...)

New Track-by-track or Album-by-album global shuffle modes It is now possible to fine tune Global shuffle and Novelties functions by selecting track by track (current default) or album by album mode (full albums are played)

Animation view pattern Animation View text is now  customizable using a pattern
 

Ambience selector It is now possible to filter selection from global functions (Novelties, Shuffle, Best-of) with selected ambience from the combo box at their left.

Catalog view enhancements
  • Catalog View now displays each cover only once (even if several albums have the same cover)
  • The number of displayed covers are now limited and a pagination sysem is now available.

New Configuration menu A new Configuration menu now agregates all the option and confguration wizards

New Debug dialog A new Show debug traces  feature available from the Help menu displays current traces.

Wikipedia view under Linux
Wikipedia view is now available under Linux

Detailed changes can be found in this changelog report
