Jajuk Brand Identity Guidelines

Introduction

To promote consistency of the visual and verbal attributes of the Jajuk software, this document will show some of the basic guidelines as where decided via the projects mailing lists. Changes and enhancements are welcomed. The contact persons regarding the Jajuk brand identity are Nicolas Schudel and Bertrand Florat.

Visual Identity

Colors

  • The main color of the Jajuk theme is light green
  • The background colors are gradients of cool gray
Lime Green:
#9cec4f
Cool Gray:
#34363b
Light Gray:
#b9b8b8

Font

The font used for the logotype is Vera Sans Mono, an open source font created by Bitstream. Vera was chosen due to its similarity with the letter J in the logo and because it is freely available.

Tahoma is a similar font and can be used instead of Vera if needed (for instance on the web page).

Logo

The Jajuk logo should be placed in its entirety onto a white or light gray background.

Download the high resolution version of the Jajuk logo or the Jajuk icon to optimize them for different media.

Verbal Identity

Project Title

The project title Jajuk should be written capitalized, for instance:

Correct:
Jajuk
Incorrect:
JaJuk, jajuk

Descriptor

The following sentence should be used in order to give the title more meaning and briefly communicate what the project is about: Software that organizes and plays music.

Tagline

In addition to the project title, a tagline can be used for promotion: Advanced Jukebox
