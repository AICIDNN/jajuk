The Information view is the lower part of Jajuk window. It's a static view that is always visible in all perspectives. It displays various jukebox information.

Information view displays:

  • A message box: most of the time, it's the name of the played track but it could be technical messages if an error occurs or if you changed configuration. Information messages are in blue and error messages in red.
  • A selection box: when you select one or more items, it shows the number of items and their size.
  • A total time: shows selection total elapsed time.
  • A current track time: shows current track elapsed time using a progress bar.
