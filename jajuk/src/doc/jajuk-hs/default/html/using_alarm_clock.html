Alarm Clock

  • This is a cool new feature which allows you set alarms to start or stop Jajuk player at a preset time.

Set new alarm

  • Select Set new alarm option from Menu, Tools, Set new alarm .
  • You will see a popup window, where you can set the "Time", "Action" (Start/Stop), "Alarm Message" and the audio file that you would like to play at the set alarm time.
  • At the set time, Jajuk will automatically (Start/ Stop) playing and also display the set "Alarm Message".

Stop an alarm

  • You can also disable/stop a set alarm from Menu, Tools, Show set alarms .
  • Menu, Tools, Show set alarms lists all the alarms which are active.
