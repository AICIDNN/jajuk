This view simply displays logical playlists. One playlist can map several playlist files, contrary to physical playlists (see view physical playlists ). It features smart playlists (Queue, Bestof, New, Bookmarks and Novelties) and regular ones. Further information about playlists on the using playlists page. See also the playlist editor at view playlist editor .

Please note that this view is hidden by default (it is accessible by moving the mouse over the left-side tab). You can make it always visible using the drawing-pin.
