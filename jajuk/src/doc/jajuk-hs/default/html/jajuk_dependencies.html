The OFFICIAL list of API, frameworks or packaging tools Jajuk includes can be found on http://jajuk.info/index.php/Jajuk_Dependencies. We would like to thanks their teams for making Jajuk possible. Note that all these libraries are included in the Jajuk package so you don't have to download or install them individually.
