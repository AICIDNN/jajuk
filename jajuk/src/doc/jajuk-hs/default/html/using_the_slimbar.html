Jajuk Slimbar

Jajuk now has a new Slim interface to control the player. It is a small control panel which can be easily placed anywhere on the desktop by dragging it.

Usage

To Switch from slimbar to full jajuk window, click on the right most button in the Jajuk Menu bar (just below the close botton)''

Features

A snapshot of Jajuk Slimbar is shown below.

  • Currently it comes with play/pause; move to next/previous songs; mute/increase/decrease volume; all the smart functions viz. Best Of, Novelties, Shuffle and Continue; Rating button and a small Search Box to quickly search and play a song.
  • The slimbar can be dragged anywhere on the screen and its position is saved between jajuk sessions
  • The current playing track is available by moving the mouse over the play/pause button
  • The current playing track ratting is available by moving the mouse over the increase rating button
  • The volume can be changed by using mouse wheel over the mute button
