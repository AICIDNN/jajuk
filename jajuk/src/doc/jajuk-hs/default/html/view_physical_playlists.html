This view simply displays all available files playlists. Each playlist file on the device is shown as one entry, contrary to logical playlists (see view logical playlists ). It also features smart playlists (Queue, Bestof, New, Bookmarks and Novelties) and regular ones. Further information about playlists on the using playlists page. See also the playlist editor at view playlist editor .

Please note that this view is hidden by default (it is accessible by moving the mouse over the left-side tab). You can make it always visible using the drawing-pin.
