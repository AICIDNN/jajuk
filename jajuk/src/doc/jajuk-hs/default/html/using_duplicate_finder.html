using duplicate finder

Find duplicate tracks

  • Get rid of duplicate audio files from you system through this new feature.
  • Select this option from Menu, Tools, Find duplicate tracks or from the right click menu on Collection node at the top in Files tree & Tracks tree views.
  • A list of duplicate tracks found on the system will be displayed in a new window. A set of duplicate tracks will be grouped together.
  • You may choose to individually select the duplicate tracks and delete them or you may use the Select all duplicate tracks button to select all duplicate tracks and delete the in one shot.
