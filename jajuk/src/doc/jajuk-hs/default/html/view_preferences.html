This view contains all the Jajuk configuration. Note you will have to press the OK button in the lower part of the view to apply changes. You can reset everything to the factory defaults by clicking on the restore defaults button.

Security notice : all passwords set here (proxy password, Last.FM password...) are *not* stored in clear in the jajuk configuration files but obfuscated

Note that few hardly used options are not accessible using the graphical interface but can be set by editing the configuration file.

Options tab

This tab contains some general options:

  • Only show available tracks [default=false]: If checked, Jajuk will become unaware of all items located on unmounted devices. It can be useful if you won't see tracks from unavailable devices.
  • Push on double click [default=false]: If checked, Jajuk default behaviour when double-clicking on a track will be to push it into the queue, not to play it.
  • Push on drop [default=true]: If checked, Jajuk default behaviour when drag and dropping a track will be to push it into the queue, not to play it.
  • Synchronize tree and table view [default=false]: If checked, table view will only display files (physical view) or tracks (logical view) from tree view selection.
  • Show a popup when launching a track [default=true]: If checked, a tray popup will be shown at the bottom of the screen when launching a track.
  • Enable Hotkeys [default=false][Windows only]: Use hotkeys so keystrokes (see list in keys shortcuts ) like F8 work even out of Jajuk.
  • Language [default=your system locale if supported or English otherwise]: Interface language. Needs a restart to be taken into account.

Modes tab

This tab contains player engine preferences:

  • Intro begin position (%) [default=0]: For intro mode, beginning track position in percent.
  • Intro length (sec) [default=20]: For intro mode, track length in seconds.
  • Cross fade duration (seconds) [default=10]: Define cross fade between two tracks. If mplayer is not available, the cross fade is not always honored
  • Best of playlist size [default=20]: Size of the bestof smart playlist (see using playlists )
  • Time limit of novelties (days) [default=30]: After this number of day since first refresh, the tracks are no more novelties and will not be chosen any more by the novelties function (Command view).
  • Number of visible planned tracks [default=10]: In Playlist editor view, number of visible planned tracks (see view playlist editor ).
  • Cross-fade duration [default=0]: Cross-fade duration in secs between two tracks. Zero means no cross-fade.

GUI tab

  • Show user interface at start-up [default=true]: Should we display the full Jajuk window or just the tray at startup ? (a single click on the tray displays the window)
  • Show balloons [default=false]: Should jajuk show a popup detailing new launched tracks ?
  • Font size [default=12]: font size used mainly in tree and table views
  • Theme [default=Orange]: Color theme to use
  • Watermarks [default=Image]: watermark to use
  • Watermark image [default=default jajuk background]: set here a custom image if you previously selected image watermark
  • Show album popups [default=true]: Should Jajuk show popups on thumbs (mainly in view catalog and view suggestions )
  • Catalog view preferences / Catalog view page size [default=200]: set here the number of albums by page in the view catalog . Zero means all in one.

Startup tab

This tab is used to select Jajuk startup mode [default=Last one at last position]:

  • Nothing : no track at all is launched at startup.
  • Last one from beginning : last track is fully played.
  • Last one at last position : last track is launched at position you stopped it when exiting Jajuk.
  • Shuffle track : A shuffle track from the entire collection.
  • Best of track : A track from your best of..
  • Novelties : A track from your novelties.
  • Track : choose a track by making a search in the text field and clicking of wanted track.

Tags tab

This tab is used to tune Jajuk tag features

  • Use parent directory as album name [default=true]: If the album tag is not available, take parent directory name as album name.
  • File Organizer Pattern : set pattern used when using the files organizer .
  • Animation Pattern : set text pattern of view animation (variables are the same that Organizer's pattern)

History tab

This tab is used to tune the History bar (in view command ). Note: the history data is in the {home directory}/.jajuk/history.xml file.

  • History duration [default=-1]: Number of days you want to keep history. -1: infinite, 0: no history.
  • Clear History button: Clears the history bar.
  • Reset ratings button: Reset all tracks ratings to zero.

Last.FM

  • Enable Last.FM information queries : Allow jajuk to make queries to Last.FM web services in order to display various information on playing artist like similar artists and others albms in the view suggestions
  • Scrobble my listening : Allow Jajuk to send information on titles you listen to Last.FM website in order to feed your Last.FM profile (check http://last.fm website)
  • Last.FM user name : Last.FM login name you used when creating your Last.FM profile
  • Last.FM password : Last.FM password you used when creating your Last.FM profile

Covers tab

This tab is used to tune covers (see view_cover ).

  • Shuffle cover [default=false]: Shuffle covers when playing a new album.
  • Load cover at each track [default=false, only if shuffle cover option is set]: allows selecting a different shuffle cover at each new track and not only at each new album.
  • Use auto cover [default=true]: Grab remote covers from the Web.
  • Preload covers [default=false, only if auto-cover option is set]: Load all found remote covers from the Web and don't wait the cover to be requested. It allows navigating more quickly into covers when all of them are loaded.
  • Minimum cover size in KB [default=5, only if auto-cover option is set]: Minimum size for a remote cover to be taken.
  • Load cover at each track in KB [default=100, only if auto-cover option is set]: Maximum size for a remote cover to be taken.

Confirmation tab

This tab is used to tune confirmations.

  • Before physically deleting a file [default=true]: Ask before physically and definitively deleting a file.
  • Before exiting Jajuk [default=false]: Ask before quitting Jajuk.
  • Before removing a device [default=true]: Ask before removing a device. No data will be deleted on your disk but the device will be deleted from the Jajuk database.
  • Before physically deleting a cover [default=true]: Ask before physically and definitively deleting a cover (image file).
  • Before clearing history [default=true]: Ask before clearing history in history bar.
  • Before reseting ratings [default=true]: Ask before reseting all tracks ratings to zero.

Network tab

This tab is used to tune network options (useful if used inside a corporate network for example)

  • Select your HTTP proxy [Default: no proxy] : No proxy HTTP proxy or SOCKS proxy
  • Proxy host name [default= proxy , only if Use a HTTP proxy is set]: Proxy host name or IP address.
  • Proxy port [default= 3128 (sometimes 8080 for some proxies), only if Use a HTTP proxy is set]: Proxy port
  • Proxy user name [no default]: Proxy user name
  • Proxy password [no default]: Proxy password
  • Connection timeout in seconds [default=5]: Timeout in seconds when establishing a connection to a server

Advanced tab

This tab is used to tune some Jajuk advanced parameters

  • Use Regexp in filters [default=false]: Use regexp strings like .*foo.* in the filter field of the Table view.
  • Backup size (MB) [default=20, only if Backup collection file option is set]: Maximum backup size of all collection file backups (collection_<date>.xml files in jajuk repository).
  • Collection charset [default=UTF-8]: Collection charset. UTF-16 is optimized for Asian languages and UTF-8 is optimized for European languages.
  • Log level [default=Error in normal mode, debug in debug mode (-debug)]: Set the verbosity of logs in your {Jajuk repository}/jajuk.log file. Please set to Debug before sending us a bug.
  • Sound engine arguments [default=none]: Additional MPlayer arguments like '-ao esd' or '-ao arts' for ie
  • Environment variables [default=nonel]: Set environment variables used by the audio player (ARTS_SERVER under KDE or ESPEAKER under Gnome for ie). Format: var1=xx var2=yyy...
  • Jajuk home path [default=user home directory]: Set the directory containing Jajuk configuration. This directory is called the Jajuk workspace , it contains one or more Jajuk repository (.jajuk directories, one for each test release and one single for final release)
  • Check Jajuk updates [default=true]: Allow Jajuk at every startup to check on the jajuk website if a new release becomes available (can be forced using the Help menu).
