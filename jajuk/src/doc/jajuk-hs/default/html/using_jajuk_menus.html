using jajuk menus

File

  • Exit (Alt-x): Quit Jajuk. Note you can also exit Jajuk by simply closing the Window or using the tray

Views

  • Restore views (current perspective) : Reorganize views of the current perspective as shipped by Jajuk team
  • Restore views (all perspectives) : Reorganize views as shipped by Jajuk team

Properties

  • Create custom property : add a custom attribute, see using custom properties
  • Remove custom property : remove a custom attribute, see using custom properties

Modes

  • Only show available tracks [Default=false]: do not show tracks on unmounted volumes (can be set in Preference view as well)
  • Show album popups [Default=true]: Show or hide album details popups when moving mouse over albums in various views (can be set in Preference view as well)
  • Synchronize tree and table view [Default=false]: whether the files or tracks table view should be synchronized with the files or tracks tree node selection (can be set in Preference view as well)
  • Repeat (Ctrl+t): check out modes in view command
  • Shuffle (Ctrl+h): check out modes in view command
  • Continue : check out modes in view command
  • intro : check out modes in view command

Tools

  • Find duplicate tracks : open the using duplicate finder
  • Set new alarm : open the using alarm clock
  • Show set alarms : see using alarm clock

Configuration

  • Preferences : open the view preferences
  • Manage DJs : open DJ Wizard, see using digital djs
  • Manage ambiences : open Ambience Wizard, see using digital djs
  • Configure web radio : see using web radio
  • Device wizard : open the Device fast creation wizard, see using devices

Help

  • Help Contents (F1): shows view help .
  • Tips of the Day : display the Tips of the Day dialog
  • Report a bug or feature request : used to send pre-formatted bug, feature request, note report
  • Show Debug Traces : display last debug logs
  • Check for Jajuk updates
  • About : show view about with credentials and system information
