Keystrokes

Most of Jajuk functions can be controled using keyboard. Keystrokes are:

Player

  • [Control-p]: Play / Pause (hotkey*)

  • [Control-s]: Stop

  • [Control-t]: Switch in and out Repeat mode

  • [Control-h]: Switch in and out Shuffle mode

  • [Alt-F9]: Seek rewind in current track (hotkey*)

  • [F9]: Play previous track (hotkey*)

  • [Shift-F9]: Play previous album (hotkey*)

  • [Alt-F10]: Seek forward in current track (hotkey*)

  • [F10]: Play next track (hotkey*)

  • [Shift-F10]: Play next album (hotkey*)

Sound controls

  • [F8]: Mute (hotkey*)

  • [Control-Down]: Increase gain (hotkey*)

  • [Control-Up]: Decrease gain (hotkey*)

Others

  • [F1]: Help

(*): hotkeys are enabled even out of Jajuk (Windows only) . This feature is disabled by default to avoid interfering with others programs. You can enable it from the Parameter view / Option Tab.

Intellipad commands

Jajuk users under Windows owning a Microft Intellipad keyboard can use media buttons (stop, pause, volume, mute, previous and next)
