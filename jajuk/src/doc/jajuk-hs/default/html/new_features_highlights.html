New Features Highlights in Jajuk 1.4

Jajuk team is pleased to present this new major upgrade that intend to bring a better user experience thanks a deeply redesigned graphical interface and new online features like Lyrics, Webradios and Last.FM support. Jajuk 1.4 code-named Aerodynamic comes with tens of impovements and new features.

Sound engine

New supported audio formats

  • Support for Monkey's Audio (.ape, .mac files)
  • Support for MPEG-2 (.mp2 files), requires mplayer 1.0pre8 or later under Linux and Mac.
  • See current Features#Supported_formats .

Interface

Brand new graphical user experience

  • Jajuk now uses the Substance look and Feel that comes with many themes and watermarks.
  • It is now possible to set a background picture of your choice (Preferences view / GUI tab)

Totally customizable interface

  • Any view can now be added in any perspective. For example, you now can add the new "Lyrics" view in the perspective you wish.
  • It is possible to add several times the same view in any perspective (can be useful, for example, to show at the same time a Cover view with the Album cover and another displaying author pictures)
  • It is now possible to reset all perspectives to default views configuration by a single click
  • All these features are available from the "Views" menu

Enhanced command bar

  • Bigger and smarter toolbar icons
  • New configurable "rate" button : easy way to increase the playing track rating. The step can be set from +1 to +10 points using the drop down button.

Simple perspective

  • Many users reported Jajuk to be too complex with too many windows, the new default perspective is now the "Simple" perspective that only display tracks table and suggestions. It intends to make Jajuk friendlier for first time users not yet familiarized with perspective and views system.

New features

Last.FM support

  • Support for audioscrobbing: when playing a track, the information is sent to Last.FM in order to feed your profile (Preferences view / Last.FM tab)
  • Support to others albums from the same artists and similar authors list from the Suggestion view (see below)

Suggestion View

  • Display Newest, Preferred* and Rarely listen albums from your collection to help you in selecting an album
  • Display albums list fetched from LAST.FM: all albums for current played track and similar artists
    • The albums you already own are displayed in bold and it is possible to launch them in a click
    • From contextual menu, it is possible to open the Last.FM page in your default web browser
    • Details over each album (including tracks list) and each artist (including album list) are displayed in a popup when mouse passes over items

Lyrics view

  • A new "Lyrics" view (retracted by default in Files and Tracks perspectives) displays current playing track lyrics
  • Contextual menu allow to copy to clipboard or to open in your browser the Lyrics page

Webradios support

  • A single click launch web radio from command bar
  • Jajuk comes with a list of default webradios
  • Radios can be added or deleted using the webradio wizard
  • Previous / Next buttons in command bar allow to switch quickly from a station to another
  • Current played radio is remembered at next Jajuk startup

Catalog view enhancements

  • Interface has been fully cleaned up
  • Details Popup on mouse pass over
  • Support drag and drop between album catalog view and others views (to launch an album for example)
  • New sorting mode by discovery date
  • Number of albums by page can be set (Preferences view / GUI tab / Catalog view preferences)
  • New "Any" filter

Tracks tree view enhancements

  • A new "discovery" filter allows to navigate into albums by discovery date (date when you got the album): less than one week, less than one month, less than 6 months and older
  • A new "Year" filter allows to navigate by albums years

Reporting feature reviewed

  • It is possible to get HTML or XML reports on any item (full collection, albums, artists, directories...)
  • HTML reports are clearer, smaller and easy to navigate through

Workspace move

  • It is now possible to set the place where Jajuk stores all collection data (indexes, ratings, thumbs, configuration...): set the new path in the preferences view / Advanced tab
  • This new feature allow to make Jajuk transportable from one computer to another (by storing the Jajuk workspace on an external USB drive with Music itself for example)

Font manager

  • It is now possible to set text size

Refreshing progress bar

  • New dialog with progress bar when adding new device or doing manual refreshes
  • Jajuk can continue to be used during refreshing (like before)

Show/hide unmounted tracks fast access

  • The show unmounted tracks option is now available from the menu bar (in Modes). It allows to show or hide tracks that cannot be currently accessed.

Check for new releases

  • Users are now informed of new Jajuk releases (can be disabled)
  • It is possible to check manually new releases availability from Help menu

Others

  • Jajuk 1.4 comes with many others smaller improvements, here is the detailed list
  • Among them:
    • Show/hide balloons option in tray
    • Del key support in playlist editor view
    • Quality agent now opens the Jajuk Trac page. Logs are automatically saved into the clipboard to ease the ticket creation
    • New Shuffle mode (Global Shuffle drop down button in Command bar): Shuffle album by album (play full albums in random order)
    • Better HTTP and SOCKS proxy support (especially for login based proxies)
