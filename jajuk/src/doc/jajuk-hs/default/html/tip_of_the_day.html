Note to Jajuk Developers: The following is used to generate the Tip of the day . See the documentation manager guide for more informations.

  • It is not possible for now to have picture embedded in the splashscreen, but it is possible in the wizard window. Make sure tips don't require image presence to make a sense. (USE: "continue mode <img...> : used to..." DO NOT USE: "click on <img> to..."
  • Please edit with care, do not leave holes but reuse existing numbers when deleting tips.
