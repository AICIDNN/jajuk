This cover view displays local (files on your disk) or remote (from the Web) covers. It can be configured in the view preferences , Cover and Network tabs.

Notes:

  • Maximum covers size is 1Mb to avoid memory issues. All local or remote covers over this size are just ignored.
  • When looking for local covers, Jajuk takes all supported (see supported formats page) image files found in the current directory but also in all directories of the track. For instance, if you play a track from your hard disk and this file is also on a mounted CD with different covers, Jajuk will display all of them. It is useful because this way, you can save covers (see later) without having to know on which device you saved them.
  • If no default cover is set (see "Default cover" action bellow), Jajuk displays first local files beginning by 'cover' or 'front'
  • Jajuk doesn't currently read covers from id3 tags

Control panel

  • Display previous cover
  • Display next cover
  • Delete physically the cover from your disk. By default, you will have a confirmation (can be set in the view preferences , Confirmations tab)
  • Save a current remote cover in the current track directory using its original name
  • + CTRL: Save a current remote cover on your disk using path and name of your choice
  • Make current cover the default cover so it will be displayed the first if you don't use the Cover shuffle option (is the default). If the cover is a remote cover not already on the disk, it will be downloaded and saved first
  • Size (L or @) : Gives size in KB of this image. A L means it 's a local image and a @ means it's a remote image
  • x/y : Gives current index over total number of covers available
  • Search type , set accuracy for remote covers (used only if you use the Auto-cover feature, this is the default). Low, medium and high accuracy performs search on both the artist and the album names. The difference is than the more the accuracy will be high, the less you will get covers but the more the chance of getting a right cover is high. Artists, Album and Title only perform searches respectively on the artist, the album and the track title.

    • Medium accuracy

    • Title

Cover view options

You can benefit from advanced cover option (see view preferences for more details). You can tell Jajuk to:

  • Enable/disable online cover search
  • Shuffle covers
  • Set minimum and maximum size for remote covers

Note you can also tune network options like proxy and connection time out if you experience problems getting remote covers (see view preferences , Network tab).

Cover views in the player perspective

In the Player perspective, you will find two cover views. By default, one is configured to display artist-related covers and the other one album-related covers. It should be the best way to display covers when playing. If you like this feature, we advise you to set the shuffle covers and Load cover at each track options.
