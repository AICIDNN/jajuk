La vue Référentiel de Playlistes

Cette vue affiche toutes les playlistes disponibles dont des playlistes spéciales (File, Bestof, Nouvelle, Signets et Nouveautés). La vue physique affiche les fichiers playliste et la vue logique affiche les playlistes logiques (une playliste logique peut référencer plusieurs playlistes physiques, voir[up.html Utiliser les playlistes]).

Notez que cette vue est masquée par défaut (elle est accessible en positionnant la souris au dessus de l'onglet de gauche). Vous pouvez la rendre toujours visible avec la punaise.

File: affiche les morceaux en train d'être joués.

Nouvelle playliste: utilisable pour créer une nouvelle playliste. Glissez-déposez y des morceaux à partir de la vue arbre ou table.

Signets: glissez-déposez y des morceaux pour retrouver rapidement vos morceaux.

Best of: contient les morceaux les plus écoutés.

Nouveautés: contient les morceaux les plus récents.
