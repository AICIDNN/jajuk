Utilisation des DJs et des ambiances

Qu'est ce qu'un DJ numérique ?

Un DJ numérique est similaire à un vrai: il génère des playlistes selon des règles (type de public, ambiance, transitions). Parcourir sa collection avec le mode Aléatoire est agréable mais pas suffisamment intelligent pour produire une vraie programmation. L'utilisation des DJs vous permettra de vous en affranchir et de la confier à votre jukebox préféré !

Qu'est ce qu'une ambiance ?

Les DJs peuvent utiliser le concept d'ambiance. Une ambiance est simplement un groupement de styles de musique. Par exemple, l'ambiance "Techno" peut grouper les styles Euro-Techno, House, Trance et Goa. Jajuk est fourni avec des ambiances presélectionnées mais vous pouvez en créer de nouvelles ou leur ajouter/enlever des styles.

Types de DJs

Trois types de DJs sont actuellement disponibles: transition, proportion ou ambiance. Il ont tous des attributs en commun.

Les attributs généraux

  • Le nom du DJ
  • La note minimale (de 1 à 4 étoiles): spécifie la qualité minimale que vous attendez des morceaux qui vont être joués (en se basant sur le système de notation interne). 1 signifie "pas de filtre" et 4 "seulement les meilleurs".
  • Période de recouvrement (secondes) entre les morceaux.
  • Unicité: un morceau peut-il être joué plusieurs fois dans une programmation ?

Les DJs de type transition

Ce type de DJ génère une playliste en utilisant des règles de transition. Par exemple, vous pouvez lui demander de jouer deux morceaux de [Rock ou Acoustique] puis trois morceaux de [Jazz ou Acid]...

Les DJs de type proportion

En utilisant ce type de DJ, vous pouvez spécifier combien de morceaux de chaque style vous désirez. Par exemple, vous pouvez lui demander de jouer 30% de [Rock ou Acoustique] et 40% de [Blues ou Folk]. Si le total est inférieur à 100%, le DJ complétera la sélection avec des morceaux aléatoires.

Les DJs de type ambiance

Ce DJ très simple joue les morceaux associés à une ambiance donnée.

Comment utiliser les DJs ?

le point d'entrée pour les DJ est le bouton DJ situé dans la vue commande ou via le menu Configuration -> Gérer les DJs.

  • Un simple clic sur le bouton DJ demandera au DJ par défaut de générer sa sélection et de la lancer. Notez que cette fonctionnalité est également accessible à partir du tray.
  • Vous pouvez sélectionner le DJ par défaut en cliquant sur la flèche à coté de l'icône et en sélectionnant un DJ s'il en existe. Notez que le nom du DJ sélectionné est disponible en tant qu'info-bulle sur le bouton.

L'assistant DJ

Ouvrez l'assistant DJ pour gérer les DJs (création/ suppression / modification) en cliquant sur le choix "Gérer les DJs" dans la liste déroulante du bouton DJ.

Création d'un DJ

  • Sélectionnez "Créer un DJ" dans le premier écran.
  • Sélectionnez le mode de fonctionnement du DJ: transition, proportion ou ambiance.
  • Saisissez les attributs généraux.
  • Saisissez les options spécifiques au mode choisi.

Sélectionner les styles pour les DJs transition ou proportion

Les DJs de type Transition ou proportion nécessite de sélectionner des styles en utilisant la fenêtre de sélection des styles (apparaît lors d'un clic sur un bouton). Vous pouvez sélectionner un ou plusieurs styles en utilisant les touches MAJ et CTRL. Si vous désirez sélectionner tous les styles d'une ambiance existante, choisissez en une dans la liste déroulante du haut de la fenêtre (Attention: selon les styles que vous avez déjà sélectionnés dans d'autres proportions ou transitions, seuls les styles restants sont disponibles).

L'assistant d'ambiance

Cet assistant est disponible à partir du menu Configuration -> Gérer les ambiances. Il permet de créer, supprimer ou modifier les ambiances.

  • Utilisez le bouton "Nouvelle ambiance" pour ajouter une ambiance. Saisissez son nom et choisissez les styles associés en utilisant la fenêtre de sélection. Notez qu'il est très facile de cloner une ambiance existante en la sélectionnant dans la liste déroulante en haut de la fenêtre de sélection de styles.
  • Utilisez le bouton "Supprimer une ambiance" pour enlever une ambiance.
  • Changez les styles d'une ambiance en cliquant sur le bouton des styles.

Comment utiliser les ambiences ?

Selectionnez une ambiance en utilisant la liste déroulante à droite du bouton DJ. Par la suite, toutes les playlistes générées par les fonctions Aléatoire, Nouveautés ou Bestof (en utilisant les trois boutons à gauche de la liste dans la vue Commande ou dans le tray) seront filtrées selon cette ambience. Séléctionnez l'ambiance 'Tous" pour supprimer ce filtre.

Astuces

  • Renseignez l'attribut style de vos fichiers pour donner toute la puissance à cette fonction. Vous pouvez par exemple utiliser la fonction de création de tag CDDB pour vous aider. Lire [editing.html Éditer les tags et attributs.]
  • Ne fixez pas une valeur trop haute au nombre d'étoiles désirées pour le DJ si votre collection est jeune ou vous risquez d'obtenir des playlistes très courtes.
  • Dans le cas des DJ de type proportion, n'utilisez pas de proportions sur des ambiences contenant trop peu de morceaux ou vous obtiendrez des playlistes très courtes également.
