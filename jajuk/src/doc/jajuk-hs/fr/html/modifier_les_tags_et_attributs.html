Il est possible de fixer des valeurs à partir:

  • De la vue table logique pour les morceaux et table physique pour les fichiers. Si vous ne voyez pas la colonne de l'attribut à modifier, cochez la dans le sélecteur de colonnes qui apparaît en cliquant sur l'îcone en haut à droite du tableau (voir [view_table.html#show_col Vue table / Sélection des colonnes)].
  • De l'assistant d'attributs pour tout élément (voir le paragraphe suivant).

L'assistant d'attributs

L'assistant d'attributs peut être utilisé pour afficher les modifier tous les attributs et tags d'un élément simple ou d'une sélection d'éléments. Il est accessible à partir:

  • Des vues table physique et logique (clic droit sur un ou plusieurs éléments ->
    Attributs
    )
  • Des vues arbre physique et logiques (clic droit sur un ou plusieurs éléments ->
    Attributs
    )
  • De la vue référentiel de playlistes (clic droit sur une playliste ->
    Attributs
    ).

Notes:

  • Dans le cas des fichiers, styles, auteurs et albums sont affichés à la fois les attributs propres (coté gauche) et les attributs des morceaux associés (coté droit) pour faciliter les modifications de tags.
  • Le nom des attributs personnels sont en bleu pour les différentier.
  • Dans le cas de selection multiples, l'assistant d'attributs n'affiche que les attributs pouvant être modifiés en communs et un champs vide signifie que les éléments selectionnés contiennent des valeurs différentes: saisissez une valeur et valider pour mettre à jour tous les éléments.

L'assistant d'attributs contient les colonnes suivantes:

  • Nom de l'attribut
    (attributs personnels en bleu).
  • Valeur de l'attribut
    (modifiable ou non).
  • Lien
  • Type

Lorsque tous les attributs sont fixés, cliquez sur Appliquer pour mettre à jour les attributs et éventuellement écrire les tags dans les fichiers musicaux.

Les tags ou meta-informations

Certains attributs sont à la fois stockés dans la collection de jajuk et dans les fichiers musicaux eux-mêmes (via des tags ID3 par exemple). Ces attributs sont: style, auteur, album, morceau, année, commentaire et ordre de lecture.

Si vous modifiez les tags via une autre application, utiliser l'option "Forcer la relecture des tags" et lancez un rafraichissement du support pour prendre en compte les modifications dans jajuk (voir [view_parameter.html#force_tags Vue Parametrages / onglet tags]).

Notez que les tags sont stockés au format Unicode (UTF-16LE) dans les fichiers musicaux. Ce format est compatible avec tous les lecteurs (Windows Media Player ou Winamp par exemple) et permet de stocker les caractères asiatiques.

La recherche de tag en ligne

Jajuk permet de modifier les tags de tous les morceaux d'un album en utilisant les base de donnée de musique en ligne CDDB (freedb). Pour cela, selectionnez dans la vue Arbre Physique un répertoire puis lancer "Obtenir les tags en ligne" via le menu contextuel.
