la vue editeur de playlistes

Retour vers la Table des matieres

Cette vue est utilisée pour visualiser ou modifier une playliste. Elle affiche la playliste sélectionnée dans la[view_playlist_repository.html vue Référentiel de Playlistes].

Les types d'éléments

Dans l'éditeur de playliste, vous trouverez différent types d'éléments:

Les morceaux standards de la playliste.

Dans la file uniquement: morceaux que l'utilisateur a sélectionnés.

Dans la file uniquement: morceaux planifiés. Affiche les morceaux à venir lorsque la sélection sera jouée et si vous utilisez le mode "continuer"(voir[pm.html la page Jouer de la musique]). C'est utile pour prévoir les morceaux suivants à la fin d'une sélection, surtout en mode aléatoire.

Dans la file uniquement: morceaux en boucle. Affiche les morceaux joués en boucle. Notez que vous pouvez avoir des morceaux hors boucle après une sélection en boucle.

La zone de contrôle

Jouer la playliste sélectionnée (sauf pour la file).

Sauver la playliste (toutes playlistes). Si c'est une playliste spéciale, Jajuk vous demandera un nom de fichier.

Supprimer un élément d'une playliste.

Ajouter un élément aléatoire.

Monter: décrémente l'index de lecture de l'élément.

Descendre: incrémente l'index de lecture de l'élément.

Effacer la playliste. Dans la file, cela consiste à arrêter la lecture et à vider la file.

Notes:

  • Vous pouvez également ajouter de nouveaux morceaux par glissez-déposez à partir des arbres et tableaux.
  • Vous pouvez selectionner les colonnes que vous désirez afficher en cliquant sur l'icone en haut à droite du tableau.

Menu contextuel

Vous disposez d'un menu contextuel sur les éléments du tableau excepté dans la file. Sélectionnez un ou plusieurs éléments (en utilisant MAJ ou ALT) puis clic doit, vous obtiendrez le menu suivant:

  • Jouer: joue les éléments sélectionnés.
  • Pousser: pousse les éléments sélectionnés.
  • Ajouter au Best Of: ajoute le morceau à la playliste Best Of.
  • Propriétés: Affiche les attributs du morceau.
