Lire de la musique

Mode de lecture général

Avant de lancer la lecture d'un morceau, vous pouvez définir le comportement par défaut de Jajuk dans la vue commande:


Fig. 01: Modes

1- Mode répétition: si sélectionné, lecture en boucle

2- Mode aléatoire: si sélectionné, lecture aléatoirement

3- Mode continu: si sélectionné, continue de jouer les morceaux suivants lorsque la sélection est terminée de jouer.

4- Mode introduction : si sélectionné, seule une partie de chaque morceau sera lue (le point de départ et le temps de lecture peuvent être définis dans la vue Paramètres)

Note : ces modes sont pris en compte uniquement si vous lancez la lecture avec la commande "Lecture". Vous pouvez toujours écraser ces modes en lançant la lecture d'un morceau avec l'option "Lecture répétée" ou "Lecture aléatoire".

La commande "Pousser", que vous trouverez dans beaucoup de menus, vous permettra de rajouter une sélection à la suite. Cela signifie que la sélection que vous poussez sera jouée lorsque que la sélection précédente sera terminée. Vous pouvez naviguer dans la liste d'attente en utilisant la liste de lecture spéciale "Liste d'attente"(Queue) (Voir fig. 02):

Fonctions spéciales

Pour éviter de sélectionner des morceaux, vous pouvez utiliser cette fonction spéciale de haut niveau depuis le panneau de commande ou depuis l'icône de notification :

Lecture aléatoire : lance un morceau aléatoirement depuis la collection complète (uniquement depuis les supports montés et prêts).

Jouer les morceaux préférés: lance un de vos morceaux préférés (définis dynamiquement suivant vos statistiques) et avec un soupçon de hasard.

Lire les nouveautés: lance aléatoirement un de vos nouveaux morceaux depuis votre collection.

Continuer l'album: intéressant si vous utilisez le mode aléatoire et que vous trouvez un album que vous voulez finir d'écouter.

Sélectionner les morceaux à lire

  • Sélectionner la perspective Logique ou Physique (voir Utiliser les perspectives et vues)

  • Sélectionner un ou plusieurs morceaux à lire. il y a plusieurs façons de sélectionner un morceau:

Utiliser la vue Arbre Physique ou Logique


Fig.04: Sélectionner des morceaux avec la vue Arbre

Dans la perspective physique:

  • Sélectionner un ou plusieurs fichiers ou répertoires (utiliser les touches CTRL et SHIFT)

Dans la perspective Logique:

  • Sélectionnez un ou plusieurs styles, auteurs, albums ou un ou plusieurs morceaux (utiliser les touches CTRL et SHIFT)

Lancer des éléments :

Vous pouvez :

  • Cliquer droit sur votre sélection, un menu popup apparaît. Sélectionnez alors 'lecture' ou n'importe quelle autre commande (Lecture répétée, Lecture aléatoire, Pousser)

  • Glisser poser la sélection (seulement un élément) pour l'ajouter à la liste de lecture (voir fig. 02)

  • Double-cliquer sur l'élément sélectionné

Utiliser la vue Table Physique ou Table Logique


Fig. 05: sélectionner des morceaux avec la vue Table

Dans la perspective physique :

  • Sélectionnez un ou plusieurs fichiers (chaque ligne est un fichier). Utilisez la touche SHIFT pour une sélection multiple.

Dans la perspective Logique:

  • Sélectionnez une ou plusieurs morceaux (chaque ligne représente une morceau). Utilisez la touche SHIFT pour une sélection multiple.

Lancer un élément:

Vous pouvez :

  • Cliquer droit sur votre sélection, un menu popup apparaît. Sélectionnez alors 'lecture' ou n'importe quelle autre commande (Lecture répétée, Lecture aléatoire, Pousser)

  • Glisser-déposer la sélection (uniquement un élément) pour l'ajouter à la liste de lecture (voir fig. 05)

  • Double-cliquer sur l'élément sélectionné

Utiliser la vue Commande


Fig.06: Sélectionner un morceau avec la vue Commande

Utiliser la boîte Recherche

La boîte Recherche permet de trouver des fichiers, morceaux, styles, auteurs, et albums... avec le texte renseigné. Le résultat apparaît dans une popup. Pour lancer le morceau désiré, cliquez dans la liste.

Utiliser la barre Historique:

La barre Historique affiche tous les morceaux que vous avez lus. Cette barre peut être effacée et configurée dans la vue Paramètres). Cliquez sur un morceau pour le lire à nouveau.

Contrôler le lecteur (lecture, pause, volume...)

 Muet : éteint/allume le son

 Morceau précédent : lit le morceau précédent dans la collection. Clic droit ou clic gauche+MAJ pour aller à l'album précédent.

 Morceau suivant : lit le morceau suivant dans la collection. Clic droit ou clic gauche+MAJ pour aller à l'album suivant.

 Retour rapide du morceau courant. Clic droit ou clic gauche+MAJ pour rejouer le morceau complet.

 Lecture / Pause du morceau courant

 Arrêter la lecture. Vide la file de lecture.

 Avance rapide du morceau en cours

 Volume. Notez que le niveau de volume est enregistré et restitué automatiquement au prochain démarrage de Jajuk. Vous pouvez changer le volume avec la molette de la souris.

 Position du morceau courant. Notez que la position du morceau courant est enregistrée. la lecture du morceau peut reprendre depuis cette position si vous cochez "Dernier morceau à la dernière position" dans le mode démarrage depuis la vue Paramètre / onglet Démarrage.

Le système de notation de Jajuk

Jajuk intègre un système de notation automatique qui apprend quels sont vos morceaux préférés en se basant sur la fréquence avec laquelle vous les écoutez. Le système actuel fonctionne de cette façon: +1 à partir de 20 secondes d'écoute, +1 si le morceau est écouté jusqu'à la fin. Notes:
  • La note est en général affichée comme un groupe d'étoiles dans les tableaux (de une à quatre étoiles). Vous pouvez accéder à la valeur précise via les info-bulles sur les étoiles.
  • Vous pouvez reinitialiser toutes les notes dans la vue Paramétrage / onglet Historique.
