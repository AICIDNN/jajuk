Cette vue fournit le système d'aide de Jajuk. Pour le moment, elle n'est disponible qu'en anglais et en français (les traducteurs volontaires sont bienvenus).

Cette vue utilise l'API JavaHelp ( <span lang="fr-FR">http://java.sun.com/products/javahelp/</span> ) et propose principalement:

  • Une partie haute pour naviguer dans la documentation (précédent/suivant), pour l'imprimer ou pour ajouter la page dans les favoris.
  • Une table des matières avec des sujets triés. Cliquez sur le sujet pour afficher l'aide correspondante.
  • Une zone de recherche pour trouver les pages traitant du terme recherché.
  • Une zone avec vos pages favorites.
