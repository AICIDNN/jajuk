La vue Commande est située en haut de la fenêtre de Jajuk. C'est une vue statique qui est toujours visible dans toutes les perspectives. Elle est utilisée pour piloter le lecteur et effectuer des recherches. Notez que vous pouvez déplacer chaque barre d'outil n'importe où dans la fenêtre (en haut ou en bas).

La boite de recherche et la barre d'historique

Fig. 02: recherche d'un morceau avec la vue commande

Utiliser la boite de recherche (1):

La boite de recherche peut être utilisée pour trouver tout fichier, morceau, style, auteur, album... pour une chaîne donnée. En résultat, un menu contextuel apparaît. Vous pouvez alors lancer un morceau en cliquant sur la liste, le morceau sera joué immédiatement.

Utiliser la barre d'historique(2):

La barre d'historique montre tous les morceaux qui ont été joués (peut être réinitialisé et configuré dans la [view_parameter.html vue Paramétrage] / onglet Historique). Cliquez sur un morceau pour le lancer à nouveau. Notez que la barre d'historique peut afficher les morceaux situés sur des supports non montés, dans ce cas, Jajuk vous proposera de les monter si disponibles.

Modes

Les modes permettent de fixer le comportement de lecture quand utilisé avant et pendant la lecture. Toutes les combinaisons de modes sont possibles. Notez que vous pouvez voir quels morceaux sont en boucle grâce aux icônes dans la vue Éditeur[view_playlist_editor.html

de Playliste
].

Mode boucle [défaut = faux]: Quand sélectionné, le morceau suivant de la sélection sera joué en boucle. Si vous l'utilisez pendant une lecture, seul le morceau courant sera joué en boucle.

Mode continuer [défaut = vrai]: Quand sélectionné, le lecteur ne s'arrêtera pas à la fin de la sélection mais continuera à jouer les morceaux suivants de la collection.

Mode introduction [défaut = faux]: Quand

sélectionné
, seule une partie de maque morceau sera jouée (la position de départ et la durée peuvent être fixés dans la [view_parameter.html vue Paramétrage] / onglet Options).

Fonctions spéciales

Pour éviter de sélectionner des morceaux, vous pouvez utiliser ces fonctions de haut niveau.

Jouer au hasard: lance une sélection de morceaux aléatoires parmi la collection complète (seulement à partir de supports montés). En utilisant le menu déroulant à droite, vous pouvez choisir le mode morceau par morceau (le défaut) ou le mode album par album.

Fonction best of: lance une sélection de vos morceaux favoris (appris avec la notation dynamique).

Jouer les nouveautés: lancer un des morceaux récemment ajoutés dans la collection.En utilisant le menu déroulant à droite, vous pouvez choisir le mode morceau par morceau (le défaut) ou le mode album par album.

Fonction continuer: permet d'arrêter le mode aléatoire et de finir de jouer un album en cours.

Fonction DJ: permet de lancer le DJ (voir [using_djs.html la page DJ])

Toutes ces fonctions spéciales peuvent être appliquées à une ambience donnée en utilisant le selecteur d'ambience. Selectionnez l'ambience "toutes"pour arrêter le filtrage.
