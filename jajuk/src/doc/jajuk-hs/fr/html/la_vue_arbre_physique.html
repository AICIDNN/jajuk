Cette vue se trouve dans la perspective Physique. Elle permet de naviguer dans la collection d'un point de vue physique sous la forme d'un arbre. Dans l'arbre, vous trouverez:

  • Les supports tels que vous les avez créés dans la
    [view_devices.html
    vue Supports
    ]
    . Les supports contiennent des répertoires.
  • Les répertoires
  • Les fichiers audio connus par Jajuk
  • Les playlistes

Notes

  • Les pochettes ne sont pas visibles dans l'arbre.
  • L'état de l'arbre est sauvé et restauré à chaque session Jajuk.

Vous pouvez sélectionner tout élément de l'arbre, les menus contextuels s'appliquent récursivement. Par exemple, sélectionnez le répertoire "ROCK" et "POP" puis clic droit et "Jouer aléatoirement": Jajuk jouera aléatoirement parmi tous les morceaux des répertoires ROCK et POP.

Glissez-déposez des morceaux

A tout moment, vous pouvez glissez une sélection et la déposer sur la playliste désirée (les icônes dans la vue[view_playlist_repository.html Référentiel de playlistes]) ou sur la vue[view_playlist_editor.html Éditeur de playlistes]. Si la playliste sélectionnée est la file, un glissez-déposez sur l'éditeur de playliste lancera immédiatement votre sélection. Notez que vous pouvez associer l'action de glissez-déposez à un pousser et non jouer via l'option correspondante dans la[view_parameter.html vue Paramétrage] / onglet options.

Menu contextuel des supports

Le menu contextuel (clic droit sur un élément sélectionné) contient les actions suivantes:

  • Configuration: Ouvre la boite de configuration du support.
  • Jouer: stoppe le morceau en cours et joue tout le support.
  • Pousser: ajoute tous les morceaux du support dans la file.
  • Jouer aléatoirement: stoppe le morceau en cours et joue tout le support au hasard.
  • Jouer en boucle: stoppe le morceau en cours et joue tout le support en boucle.
  • Monter: monte le support (voir
    [devices.html
    Utilisation des supports
    ]
    ).
  • Démonter: démonte le support (voir
    [devices.html
    Utilisation des supports
    ]
    ).
  • Forcer le rafraîchissement: force le rafraîchissement du support pour relire les tags (voir
    [devices.html
    Utilisation des supports
    ]
    ).
  • Synchroniser: resynchronise deux supports (voir
    [devices.html
    Utilisation des supports
    ]
    ).
  • Tester: vérifie la disponibilité d'un support.
  • Créer les playlistes (pas implémenté): créé les fichiers playliste récursivement.
  • Cherche les tags en ligne: Recherche les tags en ligne (CDDB) pour les morceaux situés à la racine de ce support.
  • Créer un récapitulatif: Génére un rapport en HTML ou XML
  • Attributs: Affiche les attributs.

Menu contextuel des répertoires

Le menu contextuel (clic droit sur un élément sélectionné) contient les actions suivantes:

  • Jouer: joue récursivement tous les morceaux d'un répertoire.
  • Pousser: pousse récursivement tous les morceaux d'un répertoire.
  • Jouer aléatoirement: joue la sélection aléatoirement.
  • Jouer en boucle: stoppe le morceau en cours et joue tout le répertoire en boucle.
  • Désynchroniser: un
    répertoire
    désynchronisé est affiché en orange et ne sera pas pris en compte lors de la synchronisation du support correspondant (voir
    [devices.html
    Utilisation des supports
    ]
    ).
  • Resynchroniser: a
    nnule
    la désynchronisation du
    répertoire
    (voir
    [devices.html
    Utilisation des supports
    ]
    ).
  • Créer les playlistes (pas implémenté): créé les fichiers playliste récursivement.
  • Copier (pas implémenté): copie la sélection.
  • Couper (pas implémenté): coupe la sélection.
  • Coller (pas implémenté): colle la sélection.
  • Effacer (pas implémenté): efface la sélection.
  • Ajouter aux favoris: ajoute recursivement à la playliste Bestof.
  • Cherche les tags en ligne: recherche les tags en ligne (CDDB) pour les morceaux situés dans ce répertoire.
  • Organiser les fichiers: organise les fichiers selon leurs tags (voir la [./reorg.html page de l'organisateur de fichiers]).
  • Créer un récapitulatif: Génére un rapport en HTML ou XML
  • Attributs: affiche les attributs.
