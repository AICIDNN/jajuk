Utiliser les attributs personnels

Jajuk permet d'ajouter des attributs personnels à pratiquement tous les éléments de l'interface (fichiers, playlistes, auteurs, albums...). Imaginez que vous n'êtes pas satisfaits du système de notation: créez alors votre propre attribut 'mesnotes' de type entier à appliquer aux morceaux. Autre cas: vous désirez enregistrer l'endroit ou se trouve vos CDs: créez un attribut de type chaîne de caractère appliqués aux supports. Vous désirez stocker la date à laquelle vous avez écouté un morceau: créez un attribut "madate" de type date etc...

Notes: 

  • Ces informations ne sont stockées que dans la base de donnée interne de Jajuk et NON dans les tags (ID3 par exemple) des fichiers musicaux. Par conséquent, elles ne sont accessibles qu'à partir de Jajuk et non à partir d'autres lecteurs ou jukebox. 

  • Si vous utilisez intensivement cette fonctionnalité, pensez à sauvegarder régulièrement votre collection (voir Utiliser les supports / sauvegardes et restaurations).

Comment créer un nouvel attribut ?

Dans le menu du haut de la fenêtre, sélectionnez Attributs / Créer un attribut. L'assistant de création d'attribut contient:

  • Elément: choisissez l'élément sur lequel sera appliqué l'attribut. Cela peut être un élément physique (Support, Répertoire, Fichier, Fichier Playliste) ou logique (Style, Auteur, Album, Morceau, Playliste). 

  • Nom de l'attribut: Le nom de l'attribut. Notez que certains noms sont réservés comme "name, id...". Vous obtiendrez un message d'erreur si vous tentez de les utiliser.

  • Type: Type de l'attribut. Jajuk supporte les entiers, les flottants, les chaînes de caractères, les dates et les booléens (vrai/faux).

  • Valeur par défaut (optionnel): Une valeur par défaut. en fonction du type de l'attribut, vous pourrez la fixer dans une zone de texte, via un calendrier ou via une case à cocher. Si aucune valeur par défaut n'est donnée, jajuk utilise les valeurs suivantes: Entier: 0, Flottants: 0.0, Chaînes: "", Dates: <jour courant>, Booléen: faux.

  • Format (optionnel): Seulement pour les dates pour le moment, sélectionnez un format de date prédéfini.

Cliquez sur OK pour créer l'attribut.

Comment supprimer un attribut ?

Dans le menu du haut de la fenêtre, sélectionnez Attributs / Supprimer un attribut. L'assistant de suppression d'attribut contient:

  • Elément: Choisissez l'élément duquel vous désirez supprimer l'attribut.

  • Nom de l'attribut: Le nom de l'attribut à supprimer. 

cliquez alors sur OK pour supprimer l'attribut. Notez que toutes les données pour cet attribut seront perdues. En cas de mauvaise manipulation, lisez Utiliser les supports / sauvegardes et restaurations.

Comment fixer les valeurs ?

On fixe les valeurs des attributs personnels de la même façon que pour les attributs standards, voir: Modifier les tags et attributs.
