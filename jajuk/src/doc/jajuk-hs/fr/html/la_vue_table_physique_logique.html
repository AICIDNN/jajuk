La vue Table Physique/Logique

Cette vue affiche votre collection sous la forme d'un tableau pour effectuer des actions contextuelles et de filtrage. Pour le moment, les colonnes sont prédéfinies mais dans les version futures de Jajuk, vous pourrez ajouter vos propres colonnes. La table physique affiche tous les fichiers de votre collection (donc vous pouvez voir plusieurs fois le même fichier s'il est dupliqué dans plusieurs répertoires) alors que la table logique n'affiche que les morceaux logiques: un simple morceau peut en fait correspondre à plusieurs fichiers mais il ne sera afficher qu'une fois.

Notes

  • La note est liée au morceau, pas aux fichiers. Elle est calculée de cette façon: +1 lorsqu'un morceau est lancé, +2 lorsqu'il est effectivement joué.
  • Vous pouvez n'afficher que les morceaux situé sur des supports montés en utilisant l'option "N'afficher que les supports montés" dans la

Glissez-déposez

A tout moment, vous pouvez glissez un morceau et le déposer sur la playliste désirée (les icônes dans la vue[view_playlist_repository.html Référentiel de playlistes]) ou sur la vue[view_playlist_editor.html Éditeur de playlistes]. Si la playliste sélectionnée est la file, un glissez-déposez sur l'éditeur de playliste lancera immédiatement votre sélection. Notez que vous pouvez associer l'action de glissez-déposez à un pousser et non jouer via l'option correspondante dans la[view_parameter.html vue Paramétrage]

Menu contextuel

Le menu contextuel (clic droit) sur une sélection de lignes contient les actions suivantes:

  • Jouer: stoppe le morceau courant et joue la sélection.
  • Pousser: pousse la sélection.
  • Jouer aléatoirement: stoppe le morceau courant et joue la sélection aléatoirement.
  • Jouer en boucle: stoppe le morceau courant et joue la sélection en boucle.
  • Jouer le répertoire entier (seulement dans la vue physique): stoppe le morceau courant et joue l'intégralité de son répertoire.
  • Jouer l'album entier (seulement dans la vue logique): stoppe le morceau courant et joue l'intégralité des morceaux de son album.
  • Jouer l'auteur entier (seulement dans la vue logique)
  • Attributs: Affiche les attributs.

Editer les éléments du tableau

Le tableau possède deux modes:

  • Lecture seule (défaut): un double clic sur tout élément du tableau lance le morceau et il n'est pas possible de modifier les valeurs.
  • Mise à jour: un double clic sur un élément éditable permet de modifier la valeur. Pour lancer le morceau, utilisez la première colonne (icone Jouer).

Pour basculer d'un mode à l'autre, utilisez le bouton situé dans la zone de filtrage.

Zone de filtrage

Pour filtrer les morceaux suivant des critères, sélectionnez le critère dans le boite déroulante (morceau, auteur...) puis saisissez la valeur recherchez dans la zone de texte puis tapez [Entrée] (la recherche se lance également automatiquement après un arrêt de la saisie).
