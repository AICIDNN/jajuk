Le tray

Le tray (aussi appelé "zone de notification") est un icône situé dans la barre de tâche sous Windows, KDE et Gnome et peut être utilisé pour piloter facilement Jajuk sans utiliser l'interface complète.


Fig.01

Clic gauche

Le clic gauche affiche/cache la fenêtre de Jajuk. Par défaut, Jajuk est lancé avec l'option -notaskbar pour masquer la barre de tâche quand il est caché. Vous pouvez supprimez cette option dans le script de lancement.

Clic droit

Un clic droit affiche la fenêtre contextuelle suivante:

o Afficher Jajuk au démarrage [défaut = vrai]: Si sélectionné, la fenêtre de Jajuk est affichée au démarrage. Sinon, seul le tray est visible au démarrage. Cette option peut être utile si vous utilisez le démarrage aléatoire ou si vous ne désirez pas utiliser l'interface au démarrage mais simplement lancer rapidement la musique.

 Mettre en pause le morceau courant.

 Stopper le morceau courant.

 Jouer le morceau précédent. Si possible, c'est le morceau précédent de la sélection sinon, c'est le morceau précédent trouvé dans la collection. Un clic droit ou un clic gauche+[MAJ] lance l'album précédent.

 Jouer le morceau suivant. Si possible, c'est le morceau suivant dans la sélection, sinon, c'est le morceau suivant dans la collection. Un clic droit ou un clic gauche+[MAJ] lance l'album suivant.

 Jouer une sélection au hasard : lance une sélection au hasard parmi toute la collection (seulement les morceaux situés sur les supports montés).

 Jouer le bestof: lance une sélection de morceaux favoris (appris par le système de notation automatique).

 Jouer au hasard: lance une sélection de nouveautés.

 Continuer l'album: Utile lorsqu'on utilise une sélection au hasard pour finir de jouer un album complètement.

 A propos de Jajuk: affiche la vue A propos.

 Couper/lancer le son

 Sortir de Jajuk

Utiliser la molette sur le tray

Vous pouvez diminuer ou augmenter le volume en utilisant la molette au dessus de la fenêtre contextuelle du tray.
