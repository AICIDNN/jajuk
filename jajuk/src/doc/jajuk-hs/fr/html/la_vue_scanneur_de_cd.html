Cette vue permet de construire une collection de CD de fichiers

musicaux
en quelque secondes. Il permet simplement de créer des supports de type CD de donnée (des CD de
fichiers
, pas des CD Audio!). Si vous
n'êtes
en encore
familier
avec la notion de support, merci de lire la page[devices.html Utiliser les supports]. Vous pouvez également créer des supports de type CD en utilisant la [view_devices.html vue Support] mais si vous désirez construire une collection de 50 CD de MP3, c'est *beaucoup* plus rapide d'utiliser cette vue pour scanner les CD en fournissant simplement:

  • Le nom du CD ("ROCK 1" par exemple).
  • Le chemin du CD (sauvé pour les indexations suivants: "F:" sous Windows, "/mnt/cdrom" sous Gnu/Linux par exemple).

Lancez alors l'indexation du support en cliquant sur le bouton "Scanner".
