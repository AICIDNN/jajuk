La vue Statistiques

Cette vue affiche différentes données sur votre collection. Actuellement, elle contient :

  • Un classement par style (intéressant uniquement si les meta-données de vos fichiers sont bien renseignées).
  • Un classement de taille par supports. Il permet de comparer la taille des supports utilisés.
  • La taille de votre collection par mois (le même fichier peut se trouver dans différents répertoires). Cet indicateur est utilisé pour connaître l'évolution de la taille de votre collection: il peut être très utile par exemple de savoir si un nettoyage est nécessaire.
  • Le nombre total de morceaux qui compose votre collection par mois. Il vous permet de savoir combien de morceaux différents vous avez dans votre collection, indépendamment du nombre de fichiers redondants.
