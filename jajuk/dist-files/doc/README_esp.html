¿ Qué es esto ?

Jajuk ( Just Another Jukebox | Java Jukebox ) es un organizador de música codificado en Java publicado bajo la GPL. No veas en el un reproductor, mejor como un avanzado gestor de colecciones.

Autores, contactos

Miembros y colaboradores están referidos en el diálogo 'Sobre' de Jajuk y en la página de los autores. Cualquier ayuda es bienvenida, especialmente para traducciones, documentación y desarrollo

¿Quienes son los usuarios de Jajuk?

Usuarios avanzados con un gran número de colecciones heterogeneas. Notar que Jajuk es mucho más que un reproductor y provee de un complejo interfaz que requiere un poco de aprendizaje

Requerimientos

Plataforma : Cualquier Sistema Operativo que soporte Java 1.4.2: Gnu/Linux, MS Windows, MacOSX, Solaris ...
  • Java : JRE 1.4.2 and superiores. JRE ( Java runtime environment ) puede ser obtenido desde la página de Sun Microsystems :  http://www.java.com/en/download/manual.jsp
  • Resolución de pantalla : al menos 800x600, 1280x1024 y superior recomendada
  • Configuración mínima : Intel P2 500, 128MB RAM y equivalentes.
  • Probado en GNU/Linux, MS Windows 2000 y Windows XP Professional SP2.

Objetivos

  • Proporcionar características de jukebox avanzado (sincronización, Base de datos de pistas , estadísticas...)
  • Ser independiente de la plataforma
  • Ser y permanecer siendo software libre para que cualquiera lo pueda extender en el futuro
  • Proporcionar buen rendimiento, especialmente al inicio

Compilación y javadoc

Si quieres recompilar Jajuk (optional), un script de ant is proporcionado en la raiz de la distribución:
  • Escribir 'ant' para recompilación
  • Escribir 'ant javadoc' para generar la documentación Javadoc
