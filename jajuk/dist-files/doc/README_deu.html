Was ist Jajuk?

Jajuk ( Just Another Jukebox | Java Jukebox ) ist ein in Java entwickeltes Musik-Verwaltungstool, das unter der GPL publiziert wird. Betrachten Sie Jajuk nicht nur als Player, sondern als fortgeschrittenen Kollektions-Manager.

Autoren, Kontakte

Team-Mitglieder und andere Mitwirkende sind im Jajuk 'Über Jajuk' Fenster und auf der Autoren-Seite aufgelistet. Weitere Helfer/innen sind stets willkommen, vor allem in den Bereichen Übersetzung, Dokumentation und Entwicklung.

Wer sind Jajuk Anwender?

Fortgeschrittene Anwender mit groﬂen oder zerstreuten Kollektionen. Bitte beachten Sie: Jajuk ist viel mehr als ein einfacher Player. Es bietet eine komplexe Benutzeroberfläche, die ein wenig Einarbeitungszeit verlangt.

Anforderungen

Plattform: Jedes Betriebssystem, das Java in der Version 1.4.2 unterstützt: Gnu/Linux, MS Windows, Mac, Solaris...
  • Java: JRE 1.4.2 oder höher. Das JRE (Java Runtime Environment) können Sie sich auf der Sun Microsystems Webseite herunterladen:   http://java.sun.com
  • Bildschirmauflösung: zumindest 1024x768; 1280x1024 und höher wird empfohlen
  • Minimale Anforderungen an den Rechner: Intel P2 500, 128MB RAM oder gleichwertige Systeme.
  • Getestet unter GNU/Linux und MS Windows 2000.

Ziele

  • Fortgeschrittene Jukebox Funktionen anbieten (Synchronisation, Titel-Datenbank, Statistiken, ...)
  • Plattform-Unabhängigkeit
  • Freie Software zu sein und zu bleiben, damit jeder Erweiterungen und Verbesserungen einbringen kann
  • Gute Performance bieten, vor allem beim Hochfahren

Kompilieren und Javadoc

Wenn Sie Jajuk selbst kompilieren möchten (optional), benutzen Sie das Ant Script aus dem Grundverzeichnis dieser Distribution:
  • Starten Sie 'ant' um Jajuk neu zu kompilieren
  • Starten Sie 'ant javadoc' um Javadoc Dokumentation zu generieren

Abhängigkeiten

Jajuk verwendet die folgenden APIs, wir danken den jeweiligen Teams:
  • Javalayer (http://www.javazoom.net/javalayer/javalayer.html): Der MP3 player
  • MP3info/id3 (http://www.ueberdosis.de/java/id3.html): Eine gute API für id3 V1 und V2 tags. Jajuk verwendet eine etwas veränderte Version.
  • Liquid Look and Feel (http://liquidlnf.sourceforge.net/): Das beste l&f für Swing; wird in Jajuk als Standard verwendet.
  • Kunststoff Look and Feel (http://www.incors.org/): Groﬂartiges look and feel
  • JFreechart (http://www.jfree.org/jfreechart/): API für Diagramme. In Jujuk für Statistiken verwendet.
  • JCommon (http://www.jfree.org/jcommon/): Swing Widgets Collection API, wird von JFreechart benötigt.
  • Java Help (http://java.sun.com/products/javahelp/): Ein mächtiges Online-Hilfe System.
  • Jakarta Log4j (http://logging.apache.org/log4j/docs/): Jakarta Logging API
  • TableLayout (http://www.clearthought.info/software/TableLayout/): Endlich ein brauchbares Swing Layout!
  • Systray4j (http://systray.sourceforge.net/): Zur Verwendung des Systrays; ist zwar native code, aber cool!
  • IzPack (http://www.izforge.com/izpack/): Umfangreicher, 100% reiner Java Installer
