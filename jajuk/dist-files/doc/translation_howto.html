Jajuk translation howto


Target

Anyone can translate Jajuk, no computer science is required at all. For any question or comments, please send me a mail : bertrand@florat.net
Translate current Jajuk version takes approximatly 2 hours.

How it works


Jajuk uses the standard Java internationalization (i18n) system based on properties file. Files are:
<jajuk installation path >/src/org/jajuk/i18n
          jajuk.properties
          jajuk_fr.properties
          jajuk_<any local>.properties
jajuk.properties contains default resource bunddle (english) and you have a property file for each language.


How to translate


Please translate last online Jajuk version.

1- Get  property file of your choice as a source language. Example: use jajuk.properties as english source language. You can find CVS latest version at :
http://cvs.sourceforge.net/viewcvs.py/jajuk/jajuk/src/org/jajuk/i18n/jajuk.properties
or any other property file in the directory.

2- Look at file structure
It is a simple key=value  format. Example:

ParameterView.73=Check this if you allow others people to come read music from your box
ParameterView.74=Password :
ParameterView.75=Set password access to your box. If no password is set, no one can read from it

Simply translate each value

Previous example in french:
ParameterView.73=Accepter que d'autres utilisateurs lise la musique située sur votre machine
ParameterView.74=Mot de passe :
ParameterView.75=Fixez un mot de passe à votre machine. Tant que le mot de passe n'est pas fixé, personne ne peut s'y connecter

3- Send me (  bertrand@florat.net ) your work as a property file with your local ( ex: jajuk_ru.properties for russian ). You can also add a message for the community on the forum:

4- ( Optional )If you want to test your work ( need ant to be installed ):
- Edit  <jajuk installation path >/src/org/jajuk/Main.java file.and search for line :
Messages.registerLocal("fr","Language_desc_fr"); //$NON-NLS-1$ //$NON-NLS-2$
Add after this line your own local. Example :
Messages.registerLocal("ru","Language_desc_ru"); //$NON-NLS-1$ //$NON-NLS-2$

- Copy your properties file in the <jajuk installation path >/src/org/jajuk/i18n directory

- Go in the <jajuk installation path > directory

- Rebuild Jajuk typing 'ant'

- Relaunch Jajuk and set language in the parameter view of the configuration perspective.

- Check your work.

Tips

  • Off course, do not modify keys but only values.
  • If a value contains a \, do not drop it, it means there is a space.
  • Keep all punctuation and specials characters like '[','.' etc
  • Try to keep value length near from english's one so labels will not be too long and not be visible with regular size.
