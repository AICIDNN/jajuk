Wat is het?

Jajuk (Just Another Jukebox | Java Jukebox) is een programma om muziek te sorteren. Zie het niet als een speler, maar meer als een geavanceerde manager van een verzameling. Jajuk wordt uitgebracht onder de GPL licentie.

Authors

Leden en medewerkers staan bij 'Over Jajuk' in het programma vermeld. Ze staan ook op de auteurs pagina. Hulp is welkom, speciaal voor vertalingen, handleidingen en programmeren.

Wie zijn de Jajuk gebruikers?

Gevorderde gebruikers met grote en/of verspreide verzamelingen. Merk op dat Jajuk veel meer is dan een speler alleen; de interface is complex en vergt inspanning.

Vereisten

Platform: elk OS dat Java 1.4.2 ondersteunt: Gnu/Linux, MS Windows, Mac, Solaris ...
  • Java: JRE 1.4.2 en hoger. JRE (Java runtime environment) kan gratis gedownload worden van de website van Sun Microsystems :  http://java.sun.com
  • Scherm resolutie : tenminste 1024x768. Aanbevolen 1280x1024 en hoger
  • Minimale configuratie : Intel P2 500, 128Mb RAM en vergelijkbaar.
  • Getest met GNU/Linux en MS Windows 2000.

Doelen

  • Geavanceerde jukebox mogelijkheden (synchroniseren, database van nummers, statistieken...)
  • Platform-onafhankelijk
  • Gratis software zijn en blijven, zodat iedereen dit in de toekomst kan voortzetten
  • Een goede performance, vooral bij het opstarten

Compileren en java documentatie

Als u Jajuk zelf opnieuw wilt compileren (dit hoeft dus niet), kunt u het ant script gebruikten in de root van deze distributie:
  • Typ ' ant' om opnieuw te compileren
  • Typ 'ant javadoc' om de Java documentatie te genereren

We maken gebruik van

Jajuk gebruikt deze API's, we bedanken hun teams:
  • Javalayer (http://www.javazoom.net/javalayer/javalayer.html): De MP3 speler
  • MP3info/id3 (http://www.ueberdosis.de/java/id3.html): Een goede API voor id3 V1 en V2 labels. Jajuk gebruikt een aangepaste versie voor perfs
  • Liquid Look and Feel (http://liquidlnf.sourceforge.net/): Beste l&f voor swing, dit is de standaar voor Jajuk
  • Kunststoff Look and Feel (http://www.incors.org/): Mooie interface
  • JFreechart (http://www.jfree.org/jfreechart/): API voor grafieken, hier gebruikt voor statistieken (taartdiagrammen...)
  • JCommon (http://www.jfree.org/jcommon/): Swing widgets collection API, nodig voor JFreechart
  • Java Help (http://java.sun.com/products/javahelp/): Krachtig online help systeem
  • Jakarta Log4j (http://logging.apache.org/log4j/docs/): Jakarta loggen API
  • TableLayout (http://www.clearthought.info/software/TableLayout/): Tenminste een bruikbare Swing layout!
  • Systray4j (http://systray.sourceforge.net/): Systray mogelijkheden, coole eigenschap!
  • IzPack (http://www.izforge.com/izpack/): 100% pure Java installer met alle mogelijkheden
