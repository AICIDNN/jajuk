Wat is het?

Jajuk (Just Another Jukebox | Java Jukebox) is een Java programma om muziek te sorteren. Zie het niet als een speler, maar meer als een geavanceerde manager van een verzameling.

Auteurs

Leden en medewerkers staan bij 'Over Jajuk' in het programma vermeld. Ze staan ook op de auteurs pagina. Hulp is welkom, speciaal voor vertalingen, handleidingen en programmeren.

Wie zijn de Jajuk gebruikers?

Gevorderde gebruikers met grote en/of verspreide verzamelingen. Merk op dat Jajuk veel meer is dan een speler alleen; de interface is complex en vergt inspanning.

Vereisten

Platform: elk OS dat Java 1.5 ondersteunt: Gnu/Linux, MS Windows, Mac, Solaris ...
  • Java: JRE 1.5 en hoger. JRE (Java runtime environment) kan gratis gedownload worden van de website van Sun Microsystems :  http://www.java.com/en/download/manual.jsp
  • Scherm resolutie : tenminste 800x600. Aanbevolen 1280x1024 en hoger
  • Minimale configuratie : Intel P2 500, 128Mb RAM en vergelijkbaar.
  • Getest met GNU/Linux en MS Windows 2000/XP.

Doelstellingen

  • Geavanceerde jukebox mogelijkheden (synchroniseren, database van nummers, statistieken...)
  • Platform-onafhankelijk
  • Gratis software zijn en blijven, zodat iedereen dit in de toekomst kan voortzetten
  • Een goede performance, vooral bij het opstarten

Compileren en java documentatie

Als u Jajuk zelf opnieuw wilt compileren (dit hoeft dus niet), kunt u het ant script gebruikten in de root van deze distributie:
  • Typ ' ant' om opnieuw te compileren
  • Typ 'ant javadoc' om de Java documentatie te genereren
