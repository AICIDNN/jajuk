What is it ?

Jajuk ( Just Another Jukebox | Java Jukebox ) is a 100% pure java music organizer in GPL. Don't see it as a player but better as an advanced collection manager

Authors, contacts

Main conceptor, developer : Bertrand FLORAT ( bertrand@florat.net ). Contributors are listed in the Jajuk 'About' view. Help is welcomed, especialy for traduction ( spanish ? ), documentation and Swing developments

Who are Jajuk users ?

Advanced users with large collections or owning MP3 CDs. Note that Jajuk is much more than a player and provides a complex interface that requires a bit leaning

Requirements

Plateform : Any OS which supports Java 1.4.2: Gnu/Linux, MS Windows, Mac, Solaris ...
  • Java : JRE 1.4.2 and above for running , JDK 1.4.2 and above for re-compiling and running. JRE ( Java runtime environment ) can be get for free at http://java.sun.com
  • Screen resolution : at least 1024x768, 1280x1024 and higher recommended
  • Minimum configuration : Intel P2 500, 128Mo RAM and equivalents.
  • Tested under GNU/Linux and MS windows 2000.

Goals

  • Provide advanced features
  • Be platteform-independent ( 100% pure java ) for use under Gnu/Linux, Mac, Windows or many others OS
  • Be and stay free software so anyone can extend it in future
  • Provide good performances, especially at startup

Compilation and javadoc

If you want to recompile Jajuk (optional) , an ant script is provided at distribution root:
  • Type ' ant' for recompilation
  • Type 'ant javadoc' to generate Java documentation
