What is it ?

Jajuk ( Just Another Jukebox | Java Jukebox ) is a Java music organizer published under the GPL. Don't see it as a player but better as an advanced collection manager.

Authors, contacts

Members and contributors are listed in the Jajuk 'About' view and in the authors page. Help is welcomed, especialy for translation, documentation and development

Who are Jajuk users?

Advanced users with large or scattered collections. Note that Jajuk is much more than a player and provides a complex interface that requires a bit learning

Requirements

Platform : Any OS which supports Java 1.5: Gnu/Linux, MS Windows, MacOSX, Solaris ...
  • Java : JRE 1.5 and above. JRE ( Java runtime environment ) can be obtained for free from the Sun Microsystems website :  http://www.java.com/en/download/manual.jsp
  • Screen resolution : at least 800x600, 1280x1024 and higher recommended
  • Minimum configuration : Intel P2 500, 128MB RAM and equivalents.
  • Tested under GNU/Linux and MS Windows 2000/XP.

Goals

  • Provide advanced jukebox features (synchronization, track database, stats...)
  • Be platform-independent
  • Be and stay free software so anyone can extend it in the future
  • Provide good performances, especially at startup

Compilation and javadoc

If you want to recompile Jajuk (optional), an ant script is provided at distribution root:
  • Type 'ant' for recompilation
  • Type 'ant javadoc' to generate the Javadoc documentation
