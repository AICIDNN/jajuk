What is it ?

Jajuk ( Just Another Jukebox | Java Jukebox ) is a Java music organizer published under the GPL. Don't see it as a player but better as an advanced collection manager.

Authors, contacts

Main conceptor, developer : Bertrand FLORAT ( bertrand@florat.net ). Contributors are listed in the Jajuk 'About' view. Help is welcomed, especialy for translation, documentation and development.

Who are Jajuk users ?

Advanced users with large or scattered collections. Note that Jajuk is much more than a player and provides a complex interface that requires a bit leaning

Requirements

Plateform : Any OS which supports Java 1.4.2: Gnu/Linux, MS Windows, Mac, Solaris ...
  • Java : JRE 1.4.2 and above. JRE ( Java runtime environment ) can be get for free from Sun Microsystems website :  http://java.sun.com
  • Screen resolution : at least 1024x768, 1280x1024 and higher recommended
  • Minimum configuration : Intel P2 500, 128Mo RAM and equivalents.
  • Tested under GNU/Linux and MS Windows 2000.

Goals

  • Provide advanced jukebox features ( synchronization, track database, stats... )
  • Be platform-independent
  • Be and stay free software so anyone can extend it in future
  • Provide good performances, especially at startup

Compilation and javadoc

If you want to recompile Jajuk (optional) , an ant script is provided at distribution root:
  • Type ' ant' for recompilation
  • Type 'ant javadoc' to generate Java documentation

Dependencies

Jajuk uses following APIs:
  • Javalayer (http://www.javazoom.net/javalayer/javalayer.html): The MP3 player
  • MP3info/id3 (http://www.ueberdosis.de/java/id3.html): A good id3 V1 and V2 tags API. Jajuk uses a modified version for perfs
  • Liquid Look and Feel (http://liquidlnf.sourceforge.net/): Greatest l&f for swing, is the Jajuk defaut
  • Kunststoff Look and Feel (http://www.incors.org/): Great look and feel
  • JFreechart (http://www.jfree.org/jfreechart/): Graphs API here used for stats ( pies...)
  • JCommon (http://www.jfree.org/jcommon/): Swing widgets collection API, required by JFreechart
  • Java Help (http://java.sun.com/products/javahelp/): Powerful online help system
  • Jakarta Log4j (http://logging.apache.org/log4j/docs/): Jakarta logging API
  • TableLayout (http://www.clearthought.info/software/TableLayout/): At least a useful swing layout !
  • Systray4j (http://systray.sourceforge.net/): Systray facilities, native but cool feature !
  • IzPack (http://www.izforge.com/izpack/): 100% pure Java full featured installer
