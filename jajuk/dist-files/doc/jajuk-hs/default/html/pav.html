Perspectives et vues

Définitions

  • Jajuk est divisé en perspectives et en vues. Une perspective est un ensemble de vues.
  • Une perspective est une façon d'appréhender les ressources gérées par Jajuk. Par exemple, vous êtes actuellement dans la perspective aide pour obtenir des informations et vous utiliserez la perspective physique pour naviguer dans les disques, répertoires, fichiers ou la perspective logique pour naviguer dans les styles, auteurs, albums, morceaux.
  • Une vue est une manière de présenter les informations relatives à une perspective. Par exemple, vous disposez à la fois de la vue arbre physique et table physique dans la perspective physique.
  • Il y a deux types de vues : les statiques ( commande, information et perspectives ) qui sont présentes dans toutes les perspectives et les vues flottantes qui sont liées à une perspective. Notez qu'une même vue peut se trouver dans plusieurs perspectives ( la vue couverture par exemple ).
  • Pour changer de perspective, cliquez sur la perspective de votre choix dans la vue perspective  ( voir la figure 02 ).

Perspectives Jajuk

La version actuelle de Jajuk comprend cinq perspectives  :


Fig. 02 : perspectives

1- Perspective physique : Dans cette perspective, vous voyez votre collection comme elle se trouve sur vos disques : fichiers, répertoires, fichiers m3u etc...  Toutes les actions se font sur un fichier physique identifié. Cette perspective est utile mais possède un inconvénient : si vous avez plusieurs fois le même morceau dans plusieurs supports ( par exemple, un sur votre disque dur et un autre sur un CD ), vous le verrez en double dans toutes les vues physiques. Idem pour les playlistes.

2- Perspective logique : Dans cette perspective, vous voyez votre collection selon les tags id3 : styles, auteurs, albums, morceaux et playlistes logiques ( une playliste logique associe plusieurs fichiers playlistes ). Cette perspective n'est utile que si votre collection est correctement taguée, sinon, tout sera 'inconnu'. Contrairement à la playliste physique, vous avez maintenant des morceaux, plus des fichiers. Un morceau est une association logique avec un ou plusieurs fichiers. Cela signifie que vous ne verrez qu'un seul morceau même s'il est associé à plusieurs fichiers.

3- Perspective configuration : contient les vues utiles pour configurer Jajuk ( options ) et pour créer ou modifier les supports. Un support est un ensemble physique de fichiers ( comme un répertoire, un CD, un baladeur ...).

4- Perspective statistiques : contient des graphiques présentant votre collection : nombre de morceaux, taille des supports, répartition par style et taille de la collection. D'autres statistiques seront disponibles dans les prochaines versions.

5- Perspective Aide : contient les vues d'aide et 'à propos' donnant diverses informations sur le projet et sur l'environnement d'exécution.
