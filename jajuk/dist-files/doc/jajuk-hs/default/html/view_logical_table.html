The Logical Table view

This view shows all tracks from your collection in a table and allows to perform contuextual operations and filtering. For the moment, columns (track name, album...) are static but in Jajuk future releases, you will be able to add new custom ones.

Notes:

  • Rate is related to the track, not to a single file. It is computed this way: +1 when a track is added in queue and +2 when it is played.
  • As usual, you can limit to files from mounted devices using the "show only mounted devices" option in Parameter view/ Option tab.

Drag and Drop

At any time, you can drag a single track from the table to any Playlist in the Playlist repository view or to the current Playlist Editor.

Contextual menu

Track contextual menu features:

  • Play: Stop current track and play track
  • Push: Don't stop current track but simply add the selection in the queue (see Queue playlist)
  • Play shuffle: Stop current track and play selection randomly (note that this mode is automaticaly taken when playing a track if you used Shuffle mode in command view)
  • Play repeat: Stop current track and play selection in loop (note that this mode is automaticaly taken when playing a track if you used Repeat mode in command view)
  • Set a property (not yet implemented). Set a property to this item
  • Properties (not yet implemented). Show item properties.

Filtering panel

To filter tracks following some criterias, select the criteria using the combo box (track name, author and so on), then type the filter value in the text field and type [enter]. If filter value is void, all files will be displayed.

o Filter type combo box: select the criteria you want to filter on.

o Filter value text field: type value you want to filter on.
