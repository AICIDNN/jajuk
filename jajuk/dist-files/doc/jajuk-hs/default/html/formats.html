Supported formats

Audio formats

Jajuk supports following audio formats:

  • Ogg Vorbis (.ogg)

  • MP3 (.mp3, Mpeg Layer 3)

  • AU (.au, Sun Audio)

  • AIFF (.aiff, Apple Interchange Format)

  • WAV (.wav)

  • SPEEX (.wav)

Playlist formats

Jajuk uses M3U (.m3u) Playlist format.

Cover formats

Jajuk supports following image formats for covers (limited to 500KB):

  • PNG (.png)

  • JPEG (.jpg)

  • GIF (.gif)

Tags formats

Jajuk supports following tags formats:

  • OGG tags

  • ID3 V1 tags

  • ID3 V2 tags except 2.4
