Catalog view

This view displays all your albums using cover thumbnails and allows to use your collection visually. Note that all thumbnails have to be generated at first access (takes a while). If an album has several cover, this view displays the default cover. Setting a default cover in Cover View refreshes the thumbnail in this view.

Sorting and filtering albums

  • "Sort by" combo enables album sorting by author/ style or album (default).
  • "Filter by" combo enables album dynamic filtering by character string.

Show / hide albums without cover

The "Show albums without cover" enables to show or hide albums without covers. It can be useful to see albums without covers to interact with them (play, get a cover online...).

Thumbnails size

The "Thumbnail size" combo enables to select thumb size (50x50, 100x100, 150x150(default) or 200x200).

Refresh thumbnails

The "Refresh thumbnails" button deletes all thumbnails are recreate them.

The contextual menu

Contextual menu on thumbnails features:
  • Play: Play album
  • Push: Push the album into queue
  • Play Shuffle:  Play album with shuffle
  • Play Repeat: Play album in loop
  • Grab cover online: Opens Grab cover online wizard
  • Properties: Display album properties

The Grab Covers Online wizard

This dialog allows user to perform an online query to get a matching cover. If covers are found, navigate into them using "next"/"previous" buttons and select matching cover using the "OK" button.
