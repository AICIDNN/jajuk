The Statistics view

This view display various facts about your collection. For the moment, it features:

  • The style repartition (it is reliable only if your collection is well tagged). It allow you to know which style you own the most.
  • The size repartition by devices. It allows you to know what are your device sizes in comparaison.
  • Your entire collection size per mounth (a same track can map several files). It allow you get your collection size evolution: it can be useful to know if some cleanups are required for exemple.
  • Your entire collection number of tracks per mounth. It allow you to know how many actual different tracks you have, independently of the number of redondant files you have in your collection.
