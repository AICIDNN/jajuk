The Physical/Logical Table view

These views show all files/tracks from your collection in a table and is used to perform contextual operations and filtering. For the moment, columns (track name, directory...) are static but in Jajuk future releases, you should be able to add new custom ones. Physical Table view displays all files from your collection (so you can get several time the same file if located in several devices or directories) when the Logical Table view only deals with tracks and related information. Note that a single track can actually map several files but will only be displayed once in this view.

Notes:

  • Rate is related to the track, not to a single file. It is computed this way: +1 when a track is added in queue and +2 when it is played.

  • As usual, you can limit to files from mounted devices using the "show only mounted devices" option in Parameter view/ Option tab.

Drag and Drop

At any time, you can drag a single file or track from the table to any Playlist in the Playlist repository view or to the current Playlist Editor.

Contextual menu

File/track contextual menu features:

  • Play: Stop current track/file and play it

  • Push: Don't stop current track but simply add the selection in the queue (see Queue playlist)

  • Play shuffle: Stop current track and play selection randomly (note that this mode is automatically used when playing a track if you set Shuffle mode in command view)

  • Play repeat: Stop current track and play selection in a loop (note that this mode is automatically used when playing a track if you set Repeat mode in command view)

  • Play directory (only physical table): stop current track and play all files found in the selection directory

  • Play album (only logical table): stop current track and play all tracks of the selection album

  • Play author (only logical table): stop current track and play all tracks of the selection author

  • Set a property (not yet implemented). Set a property to this item

  • Properties (not yet implemented). Show item properties.

Filtering panel

To filter tracks/files following some criteria, select the criteria using the combo box (track name, author and so on), then type the filter value in the text field and type [enter] (note that filtering occurs also automatically when typing stops). If filter value is void, all tracks/files will be displayed. "Any" criteria means that search is done on all available criteria.

o Filter type combo box: select the criteria you want to filter on.

o Filter value text field: type value you want to filter on.
