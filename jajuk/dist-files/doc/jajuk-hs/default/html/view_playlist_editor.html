The Playlist Editor view

This view can be used to watch queue states or to change tracks order, to add new tracks or remove others. It works on the playlist selected in the Playlist Repository view.

Different types of items

In the playlist editor, you will find several types of items:

 Regular tracks in the playlist. In the Queue playlist, it matches user selection to be played.

 Planned tracks (only in Queue playlist): shows next tracks to be played when selection is done and if you use the continue mode (see Playing music page). It is useful to know which are next tracks when there are no more than one or few files in the queue.

 Repeated tracks (only in Queue playlist): shows tracks that are played in a loop. Note that you can have a regular selection after a repeated one but never the opposite.

The Control Panel

 Play selected playlist (all playlists except Queue)

 Save the playlist (all playlists). If it is a special playlist, Jajuk will ask you to provide a path and filename to store the playlist.

 Add a new file to this playlist. When none line is selected, it adds a new item at the end of the playlist. Otherwise, it adds the track at the current selected line (previous item goes after it). In the Queue playlist, it is not possible to change current track so it is not possible to perform an add on the first line.

 Remove a file from this playlist. Next tracks go up in the list.

 Add a shuffle track. Same behavior than the regular adds.

 Up: decrease file playing index

 Down: increase file playing index

 Clear playlist. Under Queue playlist, it means to stop the player and void the queue.
