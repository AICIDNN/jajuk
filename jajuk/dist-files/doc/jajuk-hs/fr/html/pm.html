Jouer de la musique

Modes généraux

Avant de lancer des morceaux, vous pouvez fixer le comportement par défaut du lecteur dans la vue commande :


Fig. 01  : Modes

1- Mode répéter : La sélection est jouée en boucle

2- Mode aléatoire : la sélection est jouée au hasard

3- Mode continuer : lorsque la sélection est terminée, les morceaux suivants de la collection sont joués.

4- Mode Introduction : Seulement une partie de chaque morceau est jouée ( la position de début et le nombre de secondes sont configurables dans la vue paramètres ).


Notez que ces modes ne sont pris en compte que si vous lancez les morceaux par la commande 'jouer'. Vous pouvez les ignorer en lançant la sélection avec la commande 'Jouer en boucle' ou 'Jouer aléatoirement'.

La commande "Pousser" que vous trouverez dans la plupart des menus vous permet d'ajouter des morceaux dans la file du lecteur. Cela signifie que les morceaux que vous ajoutez seront joués lorsque la sélection précédente sera terminée. Sélectionner les morceaux

  • Allez dans la perspective physique ou logique

  • Sélectionnez un ou plusieurs  morceaux à jouer. Il y a de nombreuses façons de sélectionner un morceau :

Par la vue arbre physique ou logique



Fig. 04 : Sélection de morceaux dans l'arbre
                   
Dans la perspective physique :

  • Sélectionnez un ou plusieurs fichiers ou répertoires.


Dans la perspective logique :

  • sélectionnez un ou plusieurs styles, auteurs, albums ou morceaux.


Lancer la sélection. Vous pouvez :

  • Cliquer ( clic droit ) sur la sélection, une popup apparaîtra. sélectionnez alors 'Jouer' ou une autre commande contextuelle.

  • Glissez-déposez votre sélection ( pour seul un élément ) dans la file ( voir fig. 02 )

  • Double-clic sur un élément.



En utilisant la table physique ou logique




Fig. 05 : Sélection de morceaux dans la vue table.


Dans la perspective physique :

  • Sélectionnez un ou plusieurs fichiers (chaque ligne du tableau est un fichier ). Utilisez la touche Shift pour faire des sélections multiples.

Dans la perspective logique :

  • Sélectionnez un ou plusieurs morceaux ( chaque ligne du tableau est un morceau ). Utilisez la touche Shift pour faire des sélections multiples.


Lancer la sélection. Vous pouvez :

  • Cliquer ( clic droit ) sur la sélection, une popup apparaîtra. Sélectionnez alors 'Jouer' ou une autre commande.

  • Glissez-déposez votre sélection ( pour seul un élément ) dans la file ( voir fig. 02 )

  • Double-clic sur un élément.



En utilisant la vue commande


Fig. 06: Sélectionner un morceau par la vue commande

Zone de recherche rapide :

La zone de recherche rapide vous permet de trouver instantanément n'importe quel fichier, morceau, style, auteur, album... pour un texte. Comme résultat, une popup apparaît. Cliquez alors sur un élément. de la liste pour le lancer.

Barre d'historique :

La barre d'historique contient tous les morceaux que le lecteur a joué ( peut être effacée et configurée dans la vue paramètres ). Cliquez sur un élément pour le lancer à nouveau.


Contrôler le lecteur ( jouer, pause, volume...)


Vous pouvez contrôler le comportement de Jajuk en utilisant ces fonctions :



Fig. 07 : fonctions de contrôle


1- Muet : coupe le son

2- Morceau précédent : joue le morceau précédent dans la collection.

3- Morceau suivant : joue le morceau suivant dans la sélection.

4- Retour rapide dans le morceau courant

5- Lecture / pause du morceau courant

6- Stop. La file est alors vidée.

7- Avance rapide dans le morceau courant

8- Volume

9- Position dans le morceau courant
