La vue Référentiel de Playlistes

Cette vue affiche toutes les playlistes disponibles dont des playlistes spéciales (File, Bestof, Nouvelle, Signets et Nouveautés). La vue physique affiche les fichiers playliste et la vue logique affiche les playlistes logiques (une playliste logique peut référencer plusieurs playlistes physiques, voir Utiliser les playlistes).


Fig 01: la vue Référentiel de Playlistes

1- File: affiche les morceaux en train d'être joués.

2- Nouvelle playliste: utilisable pour créer une nouvelle playliste. Glissez-déposez y des morceaux à partir de la vue arbre ou table.

3- Signets: glissez-déposez y des morceaux pour retrouver rapidement vos morceaux.

4- Best of: contient les morceaux les plus écoutés.

5 - Nouveautés: contient les morceaux les plus récents.

6- Une playliste normale (fichier .m3u).

Menus contextuels

File

  • Sauver sous: sauve la file sous la forme de fichier m3u. Notez que le nouveau fichier m3u n'apparaîtra qu'au prochain rafraîchissement.

Nouvelle

  • Jouer: joue cette playliste

  • Sauver sous: sauve cette playliste sous la forme de fichier m3u. Notez que le nouveau fichier m3u n'apparaîtra qu'au prochain rafraîchissement.

Signets

  • Jouer: Joue cette playliste

  • Sauver sous: Sauve cette playliste sous la forme de fichier m3u. Notez que le nouveau fichier m3u n'apparaîtra qu'au prochain rafraîchissement.

Best of

  • Jouer: Joue cette playliste

  • Sauver sous: Sauve cette playliste sous la forme de fichier m3u. Notez que le nouveau fichier m3u n'apparaîtra qu'au prochain rafraîchissement.


Nouveautés

  • Jouer: joue cette playliste

  • Sauver sous: sauve cette playliste sous la forme de fichier m3u. Notez que le nouveau fichier m3u n'apparaîtra qu'au prochain rafraîchissement.

Playlistes normales

  • Jouer: joue cette playliste

  • Effacer: efface cette playliste ou toutes les fichiers playlistes correspondants si c'est une playliste logique.
