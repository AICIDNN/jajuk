Modifier les tags et attributs

Il est possible de fixer des valeurs à partir:

  • De la vue table logique pour les morceaux et table physique pour les fichiers. Si vous ne voyez pas la colonne de l'attribut à modifier, cochez la dans le sélecteur de colonnes qui apparaît en cliquant sur l'îcone en haut à droite du tableau (voir Vue table / Sélection des colonnes). 

  • De l'assistant d'attributs pour tout élément (voir le paragraphe suivant).

L'assistant d'attributs

L'assistant d'attributs peut être utilisé pour afficher les modifier tous les attributs et tags d'un élément simple ou d'une sélection d'éléments. Il est accessible à partir:

  • Des vues table physique et logique (clic droit sur un ou plusieurs éléments -> Attributs)

  • Des vues arbre physique et logiques (clic droit sur un ou plusieurs éléments -> Attributs)

  • De la vue référentiel de playlistes (clic droit sur une playliste -> Attributs).

Notes:

  • Dans le cas des fichiers, sont affichés à la fois les attributs fichiers (coté gauche) et les attributs morceau associés (coté droit) pour faciliter les modifications de tags.

  • Le nom des attributs personnels sont en bleu pour les différentier.

  • Dans le cas de sélection multiples, l'assistant d'attributs n'affiche que les attributs pouvant être modifiés en communs.
L'assistant d'attributs contient les colonnes suivantes:

  • Nom de l'attribut (attributs personnels en bleu).

  • Valeur de l'attribut (modifiable ou non).

  • Lien: Si la valeur elle-même est un élément, cliquez sur ce bouton pour afficher l'assistant d'attributs associé. 

  • Type: Type de l'attribut. 

  • Album complet (seulement pour les morceaux): Si utilisé, tout changement d'une valeur sera propagé à tous les morceaux du même album.

Lorsque tous les attributs sont fixés, cliquez sur Appliquer pour mettre à jour les attributs et éventuellement écrire les tags dans les fichiers musicaux.

Les tags ou meta-informations

Certains attributs sont à la fois stockés dans la collection de jajuk et dans les fichiers musicaux eux-mêmes (via des tags ID3 par exemple). Ces attributs sont: style, auteur, album, morceau, année, commentaire et numero de chanson.

Si vous modifiez les tags via une autre application, utiliser l'option "Forcer la relecture des tags" et lancez un rafraichissement du support pour prendre en compte les modifications dans jajuk (voir Vue Parametrages / onglet tags).

Notez que les tags sont stockés au format Unicode (UTF-16LE) dans les fichiers musicaux. Ce format est compatible avec tous les lecteurs (Windows Media Player ou Winamp par exemple) et permet de stocker les caractères asiatiques.
