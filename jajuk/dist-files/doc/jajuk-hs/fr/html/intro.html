Manuel de Jajuk

Ceci est l'aide en ligne de Jajuk. Notez que vous disposez également d'info-bulles sur tous les objets graphiques.

Si cette documentation ne répond pas à vos questions, veuillez remplir une demande de support.

Si vous trouvez un bogue, veuillez remplir un rapport de bogue ou utilisez l'assistant qualité  (Aide->Envoyer un rapport d'erreur).

Pour toute autre demande, veuillez utiliser le forum.
