Aide Jajuk


Ceci est l'aide en ligne de Jajuk. La traduction francaise n'est pas terminée, la documentation complète est accessible en Anglais (changez de langue dans la vue Parametres/ onglet Options). Notez que vous disposez d'info-bulles sur la plupart des composants graphiques pour vous aider.

Merci d'envoyer vos remarques ou questions sur le forum : http://sourceforge.net/forum/forum.php?forum_id=315361
