Aide Jajuk

Il s'agit de l'aide système de Jajuk. La traduction de la documentation en Français n'est pas encore terminée. Notez que vous disposez également d'info-bulles sur tous les objets graphiques.

Si cette documentation ne répond pas à vos questions, veuillez remplir une demande de support.

Si vous trouvez un bogue, veuillez remplir un rapport de bogue.

Pour toute autre demande, veuillez utiliser le forum.
