La vue Commande

La vue Commande est située en haut de la fenêtre de Jajuk. C'est une vue statique qui est toujours visible dans toutes les perspectives. Elle est utilisée pour piloter le lecteur et effectuer des recherches.


Fig. 01: vue Commande

Menu

Fichier -> Sortir (Alt-x): Sortir de Jajuk. Notez que vous pouvez aussi sortir de Jajuk en fermant la fenêtre ou en utilisant le tray.

Vues -> Restaurer la configuration par défaut: réorganise les vues comme elles ont été livrées par le projet.

Propriétés -> (Pas encore implémenté), sera utilisé pour créer et gérer les attributs personnels.

Mode -> Contrôles pour les modes de lecture (boucle, aléatoire, continuer, introduction).

Aide -> Contenu: affiche la vue Aide.

Aide -> A Propos: affiche la vue la vue A Propos avec la liste les informations sur le projet.

La boite de recherche et la barre d'historique


Fig. 02: recherche d'un morceau avec la vue commande

Utiliser la boite de recherche (1):

La boite de recherche peut être utilisée pour trouver tout fichier, morceau, style, auteur, album... pour une chaîne donnée. En résultat, un menu contextuel apparaît. Vous pouvez alors lancer un morceau en cliquant sur la liste, le morceau sera joué immédiatement.

Utiliser la barre d'historique(2):

La barre d'historique montre tous les morceaux qui ont été joués (peut être réinitialisé et configuré dans la vue Paramétrage / onglet Historique). Cliquez sur un morceau pour le lancer à nouveau. Notez que la barre d'historique peut afficher les morceaux situés sur des supports non montés, dans ce cas, Jajuk vous proposera de les monter si disponibles.

Modes

Les modes permettent de fixer le comportement de lecture quand utilisé avant et pendant la lecture. Toutes les combinaisons de modes sont possibles. Notez que vous pouvez voir quels morceaux sont en boucle grâce aux icônes dans la vue Éditeur de Playliste.

Mode boucle [défaut = faux]: Quand sélectionné, le morceau suivant de la sélection sera joué en boucle. Si vous l'utilisez pendant une lecture, seul le morceau courant sera joué en boucle.

Mode aléatoire [défaut = faux]: Quand sélectionné, le morceau suivant sera un morceau aléatoire de la collection. Si utilisé pendant une lecture, la sélection courante sera jouée aléatoirement.

Mode continuer [défaut = vrai]: Quand sélectionné, le lecteur ne s'arrêtera pas à la fin de la sélection mais continuera à jouer les morceaux suivants de la collection.

Mode introduction [défaut = faux]: Quand sélectionné, seule une partie de maque morceau sera jouée (la position de départ et la durée peuvent être fixés dans la vue Paramétrage / onglet Options).

Fonctions spéciales

Pour éviter de sélectionner des morceaux, vous pouvez utiliser ces fonctions de haut niveau.

 Jouer au hasard: lance une sélection de morceaux aléatoires parmi la collection complète (seulement à partir de supports montés).

 Fonction best of: lance une sélection de vos morceaux favoris (appris avec la notation dynamique).

 Jouer les nouveautés: lancer un des morceaux récemment ajoutés dans la collection.

 Fonction continuer: permet d'arrêter le mode aléatoire et de finir de jouer un album en cours.

Contrôle du lecteur (jouer, pauser, volume...)

 Coupez/démarrer le son

 Morceau précédent : lit le morceau précédent dans la collection. Clic droit ou clic gauche+MAJ pour aller à l'album précédent.

 Morceau suivant : lit le morceau suivant dans la collection. Clic droit ou clic gauche+MAJ pour aller à l'album suivant.

 Retour rapide du morceau courant. Clic droit ou clic gauche+MAJ pour rejouer le morceau complet.

 Jouer / Pauser le morceau courant.

 Stopper, la file de lecture est alors vidée.

 Avance rapide dans le morceau courant.

 Volume (jauge). Notez que le volume est sauvegardé et restauré automatiquement au démarrage suivant. Vous pouvez également fixer le volume avec la molette de la souris au dessus du menu contextuel du tray.

 Position dans le morceau courant (jauge). Notez que la position dans le morceau courant est sauvée et restaurée au démarrage suivant si vous utilisez l'option "Dernier à la dernière position" dans la vue Paramétrage / onglet Démarrage.
