The command view

The command view is located at the top of Jajuk window. It is a static view that is always visible in all perspectives. It is used to control the player and perform searches.


Fig. 01: Command view

Menu

File

     File -> Open File (Alt-f): Allows opening one or more tracks (we call them basic tracks because they may not be indexed in collection). You can choose any music files from your disks, not only those from your devices. Note however that this function is not recommended because basic tracks will show less information and will not be added in the history bar. When opened, the tracks are played immediately, not pushed.

     File -> Exit (Alt-x): Quit Jajuk. Note you can also exit Jajuk by simply closing the Window or using the tray.

Views -> Restore defaults: Reorganize views as shipped by authors.

Properties -> Not yet implemented, will be used to create and manage custom properties

Mode -> Controls for modes ( repeat, shuffle, continue, intro). Same as two-states buttons bellow.

Help

    Help -> Contents: show Help view.

    Help -> About: show the About view with credentials and system information.

Search box and history bar


Fig. 02: Selecting a track with command view

Using search box(1):

The search box can be used to find any file, track, style, author, album... for a given text. A result popup appears. Then launch the track by clicking on the list: the track will be played immediately.

Using history bar(2):

The history bar shows all tracks you played (can be cleared and configured in the Parameter view). Click on one track to launch it again. Note that tracks in the history bar may be on devices that are no more mounted. In this case, Jajuk will ask you to mount them if available.

Modes

Modes set the player behavior when selected before and during the play. All combinations of modes are allowed. Note that you can see which tracks are repeated thanks special icons in the playlist editor view.

Repeat mode [default=off]): When checked, next launched selection will be played in a loop. If it is checked during a play, only current track will repeat.

Shuffle mode [default=off]: When checked, next launched selection will be played randomly. If it is checked during a play, current selection will randomize.

Continue mode [default=on]: When checked, player will not stop when selection is played but will continue to play next albums or tracks.

Intro mode [default=off]: When checked, only a part of each track will be played (begin position and play time can bet set in the Parameter view/Options tab)

Special functions

To avoid selecting tracks, you can use these high-level special functions:

 Play shuffle: launch a shuffle track from the entire collection (only from mounted and ready devices).

 Play best of: launch one of your favorite track (learned with dynamic rate) and with a bit of random.

 Play novelties: launch one of newest tracks from your collection with a bit of random.

 Continue album: Useful when using shuffle mode and you find an album that you want to play completely.

Controlling player ( play, pause, volume...)

 Mute: stop/start the sound

 Previous Track: Listen previous track in collection. Right click or Shift + left click to play previous album

 Next track: Listen next track in selection. Right click or Shift + left click to play next album

 Fast rewind in current track

 Play / Pause current track

 Stop playing. Playing queue is then voided

 Fast forward in current track

 Volume (slider). Note that volume is saved and reset automatically at next Jajuk startup. You can change volume with the mouse wheel over tray popup as well.

 Position in current track (slider). Note that current track position is saved and startup track can restart from this position if you check the "Last track at last position" startup mode in 'the Parameter View / startup.
