Réorganisation des fichiers

Si vos fichiers musicaux sont correctement taggés mais mal rangés sur votre disque ou si vous désirez uniformiser leurs noms ou répertoires, vous pouvez utiliser la fonction "Reorganisation" accessible depuis la vue Arbre physique en sélectionnant un repértoire -> clic droit -> "Réorganiser les fichiers". Cette fonction lit les tags (ID3, OGG ou autres) de vos fichiers et les renomme selon un canevas que vous pouvez paramétrer. Par défaut: auteur/année-album/no piste-nom morceau.
Vous pouvez modifier ce canevas en allant dans la vue paramétrage / onglet Tags.
Attention: cette fonction ne se lancera que sur les fichiers contenant tous les tags inclus dans le canevas.

Les champs disponibles

Pour constituer le canevas, nous pouvez utiliser les champs suivants:
  • %artist : Nom de l'auteur
  • %album: Nom de l'album
  • %genre: Style du morceau
  • %year: Année du morceau
  • %track: Nom du morceau
  • %n: Numero de piste
