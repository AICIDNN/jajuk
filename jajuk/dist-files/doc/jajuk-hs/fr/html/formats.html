Formats supportés

Formats Audio

Jajuk supporte les formats de fichier audio suivants :

Format Extension
MP3 .mp3
OGG .ogg
FLAC .flac
AAC .aac, .m4a
WAV, SPEEX .wav
AU, AIFF .au, .aiff
RealAudio .ram
WMA .wma

Formats des playlistes

Jajuk utilise le format de playliste M3U (.m3u).

Formats des pochettes

Jajuk supporte les formats d'image suivants (taille limitée à 1Mo):

  • PNG (.png)

  • JPEG (.jpg)

  • GIF (.gif)

formats de tags

Jajuk supporte les tags suivants:

  • Tags OGG

  • Tags Flac

  • Tags ID3 V1

  • Tags ID3 V2

  • Tags WMA
