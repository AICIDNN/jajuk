La vue Catalogue

Cette vue affiche l'ensemble de vos albums par vignettes et vous permet d'utiliser votre collection d'une façon très visuelle. Notez que toutes les vignettes ont à être générées à la première utilisation, ce qui prend un certain temps. Si un album possède plusieurs couvertures, cette vue affiche la couverture par défaut. Lorsque la vue par défaut est modifiée via la vue Couverture, la vignette est mise à jour dans cette vue.

Tri et filtrage des albums

  • La boîte de dialogue "Trier par" permet de trier les albums par style, auteur ou album (défaut).

  • La boîte de dialogue "Filtrer par" permet de filtrer dynamiquement les albums.

Montrer / masquer les albums sans couvertures

L'option "Montrer les albums sans couverture" permet d'afficher ou de masquer les albums sans couvertures, ce qui peut être utile pour pouvoir tout de même interagir avec les couvertures sans couverture (Lancer, obtenir une couverture en ligne...).

Taille des vignettes

La boîte de dialogue "Taille des vignettes" permet de sélectionner la taille des vignettes (50x50, 100x100, 150x150(défaut) ou 200x200 pixels).

Rafraîchir les vignettes

Le bouton "Rafraîchir les vignettes" permet d'effacer toutes les vignettes et de les recréer.

Le sélecteur de page

Si le nombre d'album est trop grand, le résultat est découpé en plusieurs pages que vous pouvez atteindre en utilisant les boutons de naviguation ou directement à partir de la boite déroulante.

Le menu contextuel

Le menu contextuel sur les vignettes comprend:

  • Jouer: Joue l'album

  • Pousser: Pousse l'album dans la file

  • Jouer aléatoirement: Joue l'album aléatoirement

  • Jouer en boucle: Joue l'album en boucle

  • Obtenir une couverture en ligne: Ouvre l'assistant de recherche de couvertures en ligne

  • Attributs: Affiche les attributs de l'album

L'assistant de recherche de couvertures en ligne

Cet assistant permet d'effectuer des recherches de couverture en ligne. Si des couvertures sont trouvées, naviguez dans la liste en utilisant les boutons "précédent"/"suivant" et valider par le bouton "OK".
