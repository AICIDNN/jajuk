La vue Animation

Cette vue est simplement un affichage du nom du morceau courant avec des effets sur le texte. Notez que la taille du texte est modifiée dynamiquement quand vous redimensionnez la vue ou lorsque vous la réorganisez.

Notez que vous pouvez modifier le texte à afficher via le champs "Format de la vue Animation" dans la vue Paramétrage / onglet Tags.
