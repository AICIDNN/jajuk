Raccourcis clavier

Une grande partie des fonctions de Jajuk peuvent se faire au clavier. Les raccourcis sont les suivants:

Contrôle du lecteur

  • [Control-p]: Jouer / Pauser (hotkey *)
  • [Control-s]: Stopper
  • [Control-t]: Passer en mode boucle / Sortir du mode boucle
  • [Control-h]: Passer en mode aléatoire / Sortir du mode aléatoire
  • [Alt-F9]: Parcourir le morceau en retour rapide (hotkey *)
  • [F9]: Passer au morceau précédent (hotkey *)
  • [Maj-F9]: Passer à l'album précédent (hotkey *)
  • [Alt-F10]: Parcourir le morceau en avance rapide (hotkey *)
  • [F10]: Passer au morceau suivant (hotkey *)
  • [Maj-F10]: Passer à l'album suivant (hotkey *)

Contrôle du son

  • [F8]: Passer en mode muet (hotkey *)
  • [Control-Bas]: Diminuer le gain (hotkey *)
  • [Control-Haut]: Diminuer le gain (hotkey *)

Autres

  • [F1]: Aide
(Hotkey *): Ces raccoursis sont disponibles même lorsque Jajuk n'a pas le focus  (Windows seulement). Par défaut, cette fonctionalité est débrayée pour éviter les interférences avec les autres programmes. Vous pouvez l'enclencher dans la vue Parametres / onglet Options.

Commandes Intellipad

Les utilisateurs de Jajuk sous Windows possédant un clavier Microsoft Intellipad peuvent utiliser les boutons multimedia pour piloter Jajuk (stop, pause, volume, muet, précédent et suivant)
