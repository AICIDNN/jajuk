Utiliser les attributs personnels

Jajuk permet d'ajouter des attributs personnels à pratiquement tous les éléments de l'interface (fichiers, playlistes, auteurs, albums...). Imaginez que vous n'êtes pas satisfaits du système de notation: crééez alors votre propre attribut 'mes notes' de type entier à appliquer aux morceaux. Autre cas: vous désirez enregister l'endroit ou se trouve vos CDs: creez un attribut de type chaîne de caractère appliqués aux supports. Vous désirez stocker la date à laquelle vous avez ecouté un morceau: crééez un attribut "madate" de type date aetc...

Notes: 

  • Ces informations ne sont stockées que dans la base de donnée interne de Jajuk et NON dans les tags (ID3 par exemple) des fichiers musicaux. Par conséquent, elles ne sont accessibles qu'à partir de Jajuk et non à partir d'autres lecteurs ou jukebox. 
  • Si vous utilisez intensivement cette fonctionnalité, pensez à sauvegarder régulierement votre collection (voir Utiliser les supports / sauvegardes et restaurations).

Comment créer un nouvel attribut ?

Dans le menu du haut de la fenêtre, selectionnez Attributs / Créer un attribut. L'assistant de création d'attribut contient:

  • Element: choisissez l'element sur lequel sera appliqué l'attribut. Celà peut être un element physique (Support, Repertoire, Fichier, Fichier Playliste) ou logique (Style, Autheur, Album, Morceau, Playliste). 

  • Nom de l'attribut: Le nom de l'attribut. Notez que certains noms sont reservés comme "name, id...". Vous obtiendrez un message d'erreur si vous tentez de les utiliser.

  • Type: Type de l'attribut. Jajuk supporte les entiers, les flottants, les châines de caractères, les dates et les booléens (vrai/faux).

  • Valeur par défaut (optionnel): Une valeur par défaut. en fonction du type de l'attribut, you pourrez la fixer dans une zone de texte, via un calendrier ou via une case à cocher. Si aucune valeur par défaut n'est donnée, jajuk utilise ces valeurs: Entier: 0, Flottants: 0.0, Chaînes: "", Dates: <jour courrant>, Booléen: faux.

  • Format (optionnel): Seulement pour les dates pour le moment, selectionnez un format de date prédéfini.

Cliquez sur OK pour créer l'attribut.

Comment supprimer un attribut ?

Dans le menu du haut de la fenêtre, selectionnez Attributs / Supprimer un attribut. L'assistant de suppression d'attribut contient:

  • Element: Choisissez l'élément duquel vous désirez supprimer l'attribut.

  • Nom de l'attribut: Le nom de l'attribut à supprimer. 

cliquez alors sur OK pour supprimer l'attribut. Notez que toutes les données pour cet attribut seront perdues. En cas de mauvaise manipulation, lisez Utiliser les supports / sauvegardes et restaurations.

Comment fixer les valeurs ?

On fixe les valeurs des attributs personnels de la même façon que pour les attributs standards, voir: Modifier les tags et attributs.
