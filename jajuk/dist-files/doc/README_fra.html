Qu'est-ce que Jajuk ?

Jajuk ( Just Another Jukebox | Java Jukebox ) est un Jukebox MP3 java et libre (GPL). Ce n'est pas un simple lecteur mais un système avancé d'organisation de votre collection.

Auteurs, contacts

Les membres et contributeurs sont listés dans la vue 'à propos' de Jajuk et sur la page des auteurs. Toute aide (développement, traduction, documentation) est appréciée.

Qui sont les utilisateurs de Jajuk ?

Des utilisateurs possédant une grande collection ou du moins une collection éparpillée de MP3. Notez que Jajuk propose de nombreuses fonctionnalités et demande un petit apprentissage (voir l'aide).

Pré-requis

Plate-forme : Tout système d'exploitation supportant Java 1.4.2: Gnu/Linux, MS Windows, Mac, Solaris, AIX ...

  • Java : JRE 1.4.2 et supérieur. Le JRE ( Java Runtime Environment ) peut être téléchargé gratuitement sur le site de Sun Microsystems : http://java.sun.com

  • Résolution : au moins 1024x768, 1280x1024 et supérieur recommandé

  • Configuration minimale : Intel P2 500, 128Mo RAM et équivalents.

  • Testé sous GNU/Linux et MS Windows 2000.

Objectifs

  • Fournir des fonctionnalités avancées ( synchronisation, indexation de supports, stats ... )

  • Etre indépendant de la plate-forme

  • Etre et rester un logiciel libre pour assurer sa pérennité et qu'il puisse continuer à être développé par quiconque.

  • Fournir de bonnes performances, principalement au démarrage.

Compilation et javadoc

Si vous voulez recompiler Jajuk (optionnel), un script ant est fourni dans la distribution :

  • Tapez 'ant' pour recompiler

  • Tapez 'ant javadoc' pour générer la javadoc

Dépendances

Jajuk utilise les API ou applications suivantes et remercie leurs équipes :

  • Javalayer (http://www.javazoom.net/javalayer/javalayer.html): Le lecteur MP3

  • MP3info/id3 (http://www.ueberdosis.de/java/id3.html): Le gestionnaire de tags id3. Jajuk utilise une version modifiée pour les performances.

  • Liquid Look and Feel (http://liquidlnf.sourceforge.net/): Le thème par défaut

  • Kunststoff Look and Feel (http://www.incors.org/): Un thème sympa

  • JFreechart (http://www.jfree.org/jfreechart/): Graphiques pour les stats

  • JCommon (http://www.jfree.org/jcommon/): Outils Swing, requis par JFreeChart

  • Java Help (http://java.sun.com/products/javahelp/): Puissant système d'aide en ligne

  • Jakarta Log4j (http://logging.apache.org/log4j/docs/): API de traces Jakarta

  • TableLayout (http://www.clearthought.info/software/TableLayout/): Enfin un bon layout swing !

  • Systray4j (http://systray.sourceforge.net/): Outils pour le systray, natif mais sympa !

  • IzPack (http://www.izforge.com/izpack/): Installer 100% pure Java
