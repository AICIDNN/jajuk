Qu'est-ce que Jajuk ?

Jajuk ( Just Another Jukebox | Java Jukebox ) est un Jukebox MP3 java et libre (GPL). Ce n'est pas un simple lecteur mais un système avancé d'organisation de votre collection.

Auteurs, contacts

Les membres et contributeurs sont listés dans la vue 'à propos' de Jajuk et sur la page des auteurs. Toute aide (déloppement, traduction, documentation) est appréciée.

Qui sont les utilisateurs de Jajuk ?

Des utilisateurs possédant une grande collection ou du moins une collection éparpillée. Notez que Jajuk propose de nombreuses fonctionnalités et demande un petit apprentissage (voir l'aide).

Prérequis

Plate-forme : Tout système d'exploitation supportant Java 1.5: Gnu/Linux, MS Windows, Mac, Solaris, AIX ...

  • Java : JRE 1.5 et supérieur. Le JRE ( Java Runtime Environment ) peut êre téléchargé gratuitement sur le site de Sun Microsystems : www.java.com

  • Résolution : au moins 800x600, 1280x1024 et supérieur recommandé.

  • Configuration minimale : Intel P2 500, 128Mo RAM et équivalents.

  • Testé sous GNU/Linux , MS Windows 2000/XP et Mac OS X.

Objectifs

  • Fournir des fonctionnalitées avancées ( synchronisation, indexation de supports, stats ... )

  • Etre indépendant de la plate-forme

  • Etre et rester un logiciel libre pour assurer sa pérénité et qu'il puisse continuer à être développé par quiconque.

  • Fournir de bonnes performances, principalement au démarrage.
