Vad är det ?

Jajuk ( Just Another Jukebox | Java Jukebox ) är en musikorganisatör i Java publiserad under GPL. Se den inte som en spelare utan hellre som en avancerad samlingshanterare.

Författare, kontakt

Medlemmar och de som bidragit finns listade i Jajuks 'Om' och på författarens sida. Hjälp välkomnas, särskilt vad gäller översättning, dokumentation och utveckling.

Vilka är användare av Jajuk ?

Avancerade användare med stora eller splittrade samlingar. Värt att notera är att Jajuk är mycket mer än en spelare och att den har ett komplext gränssnitt som kräver lite träning.

Krav

Plattform : Vilket OS som helst som stödjer Java 1.4.2: Gnu/Linux, MS Windows, Mac, Solaris ...
  • Java : JRE 1.4.2 eller senare. JRE ( Java runtime environment ) kan fås fritt via Sun Microsystems website :  http://java.sun.com
  • Skärmupplösning : minst 1024x768, 1280x1024 eller högre rekommenderas
  • Minimal konfiguration : Intel P2 500, 128MB RAM eller motsvarande.
  • Testad i GNU/Linux och MS Windows 2000.

Mål

  • Tillhandahålla avancerade jukebox-egenskaper (synkronisering, låtdatabas, statistik... )
  • Vara plattformsoberoende
  • Vara och förbli gratis programvara så att vem som helst kan utvidga den i framtiden
  • Ge goda prestanda, särskilt vid uppstart

Kompilering och javadoc

Om du vill kompilera om Jajuk (inte ett krav) , så finns ett ant-skript i distributionens root:
  • Skriv ' ant' för att kompilera om
  • Skriv 'ant javadoc' för att generera Java-dokumentation

Beroenden

Jajuk använder följande APIs, vi riktar ett tack till deras utvecklingsteam:
  • Javalayer (http://www.javazoom.net/javalayer/javalayer.html): MP3-spelaren
  • MP3info/id3 (http://www.ueberdosis.de/java/id3.html): Ett bra API för id3 V1 och V2 taggar. Jajuk använder en omarbetad version
  • Liquid Look and Feel (http://liquidlnf.sourceforge.net/): Den bästa 'look & feel' som finns till swing, är Jajuks grundinställning
  • Kunststoff Look and Feel (http://www.incors.org/): Bra 'look and feel '
  • JFreechart (http://www.jfree.org/jfreechart/): API för grafer, används för statistik (pajdiagram)
  • JCommon (http://www.jfree.org/jcommon/): API med samling av swing 'widgets', behövs av JFreechart
  • Java Help (http://java.sun.com/products/javahelp/): Kraftfullt online-hjälpsystem
  • Jakarta Log4j (http://logging.apache.org/log4j/docs/): API för loggning
  • TableLayout (http://www.clearthought.info/software/TableLayout//): Till slut! En användbar swing-layout !
  • Systray4j (http://systray.sourceforge.net/): Systray-verktyg, 'native' men cool!
  • IzPack (http://www.izforge.com/izpack/): Installeringsverktyg, 100% ren Java
