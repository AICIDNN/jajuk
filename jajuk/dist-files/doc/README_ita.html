Che cos'e' ?

Jajuk ( Just Another Jukebox | Java Jukebox ) e' uno programma fatto in Java per organizzare la musica ed e' rilasciato sotto licenza GPL. Non lo considerare un semplice lettore ma come uno strumento per organizzare le tue collezioni di brani.

Autori, contatti

Membri e contributi sono mostrati sotto il menu' 'A proposito' di Jajuk e nella pagina autori. Qualsiasi aiuto e' benvenuto, specialmente per le traduzioni, documentazione e sviluppo

Chi sono gli utenti di Jajuk ?

Utenti avanzati con ampie collezioni di brani musicali. Notate che Jajuk e' molto piu' di un semplice lettore e quindi la sua interfaccia e' un po' piu' complessa e richiede un po di studio

Requisiti

Piattaforma : Qualsiasi sistema operativo che supporta Java 1.4.2: Gnu/Linux, MS Windows, Mac, Solaris ...
  • Java : JRE 1.4.2 o superiore JRE ( Java runtime environment ) puo' essere scaricata liberamente da Sun Microsystems website :  http://java.sun.com
  • Risoluzione dello schermo : al meno 800x600, 1280x1024 o superiore raccomandata
  • Minima configurazione : Intel P2 500, 128MB RAM o equivalenti.
  • Testato sotto GNU/Linux e MS Windows 2000.

Obiettivi

  • Fornire funzioni di un jukebox avanzato ( sincronizzazione, database dei brani, statistiche... )
  • Essere indipendente dalla piattaforma
  • Essere e rimanere un software libero cosi che chiunque possa lavorarci in futuro
  • Fornire buone performance, specialmente alla partenza

Compilazione e javadoc

Se volete ricompilare Jajuk (opzionale) , uno script ant e' fornito nella radice della distribuzione:
  • Scrivere ' ant' per la ricompilazione
  • Scrivere 'ant javadoc' per generare la documetazione Java
