Jajuk Installation


This documentation targets : Gnu/Linux, MS Windows, MAC, ...

1- Requirements

See REQUIREMENTS part in the Readme file at : http://jajuk.sourceforge.net/README.html

2- Download

Download Jajuk last distribution at : http://sourceforge.net/projects/jajuk/

3- Installation

    We use IzPack Java installer ( http://www.izforge.com  ) to stay GPL and platteform-independant. The distribution file is an executable jar file, the java version of a 'setup.exe' file but working with any operating system.

        Please :

  1. Double-clic on the downloaded file ( jajuk-dist-<version>.jar ). If nothing appends, it means jar extention is not mapped to 'java' program. Then, you can map it with you OS ( open with... )  or launch it by typing in a console:   java -jar jajuk-dist-<version>.jar
  2. Select your language for installation
  3. Read carefully the provided README file
  4. Read and accept GPL licence
  5. Select installation target : Example: /opt/jajuk under Unix,  c:\Program Files\jajuk under Windows.
  6. Create shorcuts ( only for Windows ) : just clic on 'create shortcuts' buttons.
  7. (Optional) Create an automatized installation script if you want to achieve fast installations with always the same parameters.

4- First launch

Under Unix : Create a shortcut or lauch script : <jajuk installation path>/jajuk.sh
Under Windows :Launch Jajuk by clicking on Jajuk icon : Start -> Jajuk -> Jajuk

5- Uninstallation

If Jajuk doesn't address your needs, you can uninstall it with 'Uninstall' icon next to 'Jajuk ' one ( Windows only ). Under Unix, launch the following jar : <jajuk installation path>/Unistaller/uninstaller.jar
