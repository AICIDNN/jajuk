Release notes

Last update : 2005-04-02

Changes: http://jajuk.sourceforge.net/changes-report.html#0_3_3

  • [Mandrake and Suse users using the RPM] IMPORTANT NOTE FOR UPGRADE FROM 0.3.1 : due to a bug in 0.3.1 packaging, please use  rpm -Uvh --nopostun jajuk-{release}-suse.noarch.rpm or uninstall 0.3.1 and install newer release.

  • [Mandrake users using the RPM] Jajuk icon should appear in Multimedia/Audio section. It can take a while for the icon to be visible. If you get no icon after one minute, please open a shell and launch menu-updates as your regular user.

  • [Windows users using exe package] We have changed the Self Extracting tool (we now use 7-ZIP), so this distribution should no more be seen as infected by some anti-virus scanners.

  • [Windows users using exe package] In some seldom cases, you may encounter a "IzPack launcher failed" message at the end of the installation. Just ignore this message, your install is safe.

PKG

Packaging tests : install Jajuk in several languages. Target directory : /prog/jajuk and c:\Program files\jajuk. Version overwriting tests : install Jajuk on existing release directory, then  remove the directory and install it to this unexisting location. Start and close Jajuk at each test.




SRT

Tests with existing collection : start jajuk, lauch some tracks.




SCR

Tests from scratch. Remove .jajuk directory, create a directory device with auto-refresh, lauch some tracks and then quit Jajuk and start it again, launch some tracks, quit and test again.




FOR

Tests on formats : launch at least one mp3, ogg, au, aiff and wav file.




DEV

Devices tests : create a directory device, refresh it, unmount it, mount it, remove it.




SYN

Device synchronization : test with a directory device and a player device, if possible, try to mix FS : ext3 for directory, FAT32 for player.




SRC

Instant search box : perform some searches and launch tracks.




HST

History : check all launched tracks are OK. Clear history and launch others tracks, check it again.




COP

Compile and test the installed jajuk sources




SUP

Startup tests : change startup option and test it.




BUP

Backup tests : change backup options and test them.




LNG

Test Jajuk in English and at least with 2 others langpacks, check translation for untranslated fields if any.




LNF

Change Look and Feel, test it.




STA

Check stats




TIM

Check elapsed / remaining time in information bar




DOC

Check About and online doc in all languages.




COM

Command panel tests : rewind, forward, volume...




PLA

Physical and logical playlists tests.




COV

Tests covers display and manipulation




VIE

Change views size / position, close jajuk and restart, check position is saved. Reload default perspectives in Parameter view / Perspective tab, test.




RES

Change resolution (1024x768 to 1600x1400) and check everything is OK




ERR

Rename or delete physically some files in your collection without refreshing device to simulate read errors, check behavior (error message in information panel, history bar...)




TRA

Tray tests

Random tests, general use tests in every views.
