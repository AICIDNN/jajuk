Què és això ?

Jajuk ( Just Another Jukebox | Java Jukebox ) es un organitzador de música programat en Java publicat sota la GPL. No vegis només un reproducto, millor mira'l com un avançat gestor de col·leccions.

Autors, contactes

Membres i colaboradors están referits en el diàleg 'Sobre' de Jajuk i a la pàgina dels autors. Qualsevol ajuda és benvinguda, especialment per a traduccions, documentació y desenvolupament

Qui són els usuaris de Jajuk?

Usuaris avançats amb un gran número de col·leccions heterogèneas. Notar que Jajuk és molt més que un reproductor i proveeix d'una complexa interfaç que requereix una mica d'aprenentatge

Requeriments

Plataforma : Qualsevol Sistema Operatiu que soporti Java 1.5: Gnu/Linux, MS Windows, MacOSX, Solaris ...
  • Java : JRE 1.5 i superiors. JRE ( Java runtime environment ) es por obtenir de la pàgina de Sun Microsystems :  http://www.java.com/en/download/manual.jsp
  • Resolució de pantalla : almenys 800x600, 1280x1024 y superior recomenada
  • Configuració mínima : Intel P2 500, 128MB RAM y equivalents.
  • Provat en GNU/Linux, MS Windows 2000 y Windows XP Professional SP2.

Objectius

  • Proporcionar característiques de jukebox avançat (sincronització, Base de dades de pistes , estadístiques...)
  • Ser independent de la plataforma
  • Ser i romandre siguent software lliure per a que qualsevol el pugui extendre en el futur
  • Proporcionar bon rendiment, especialment a l'inici
