这是什么？

Jajuk( 另外一个点唱机 |Java 点唱机 ) 是一个发布于 GPL 的 Java的音乐管理软件。千万别把它单单看作一个播放器，它更是一个功能强大的音乐搜藏与管理软件。

作者，联系

Jajuk的’About’菜单和作者网页中列出了制作成员以及贡献者的名单。任何形式的帮助都将受到欢迎，特别是翻译、文档以及开发工作。

谁是 Jajuk 的用户？

拥有大量或零散音乐收集的高级用户。因为Jajuk提供了复杂的界面，功能远远不局限于简单的播放器，从而需要用户花一定的时间学习使用。

配置要求

系统平台：能够支持 Java 1.4.2 的任何系统： Gnu/Linux,MS Windows, Mac, Solaris……

  • Java: JRE 1.4.2 或更高版本。 JRE(Java runtime environment) 可以从 Sun Microsystems 的网站免费获得 : http://java.sun.com
  • 显示分辨率：1024x768 以上，推荐 1280x1024 或更高
  • Intel P2 500, 128Mo RAM 或同等配置
  • 在 GNU/Linux 和 MS Windows 2000 环境下通过测试

目标

  • 提供高级的音乐管理功能 （同步处理，音轨数据库，数据……）
  • 跨平台运行
  • 成为并保持为自由软件，这样将来任何人都可以对其进行扩充
  • 提供优异的性能，特别是启动的效率

如果你想要重新编译 Jajuk( 可选 ) ，在分布的目录下有一个 ant 脚本：

依赖的技术

Jajuk 使用了一下的 API ，感谢这些制作小组：
  • Javalayer (http://www.javazoom.net/javalayer/javalayer.html): MP3 播放器
  • MP3info/id3(http://www.ueberdosis.de/java/id3.html):一个优秀的id3V1和V2tagsAPI。Jajuk使用了他们产品的一个改进版本
  • Liquid Look and Feel (http://liquidlnf.sourceforge.net/): 最好的 swing l&f ，是 Jajuk 的默认配置
  • Kunststoff Look and Feel (http://www.incors.org/): 良好的视觉效果
  • JFreechart (http://www.jfree.org/jfreechart/): 图形 API 用来实现数据（饼图等……）
  • JCommon (http://www.jfree.org/jcommon/): Swing widgets collection API, JFreechart 所必需的
  • Java Help (http://java.sun.com/products/javahelp/): 强大的在线帮助系统
  • Jakarta Log4j (http://logging.apache.org/log4j/docs/): Jakarta logging API
  • TableLayout (http://www.clearthought.info/software/TableLayout//): 至少是一个有用的 swing layout ！
  • Systray4j (http://systray.sourceforge.net/): Systray 便捷工具 , 简单实用 !
  • IzPack (http://www.izforge.com/izpack/): 100% 纯正的 Java 安装工具
