Linux Installation Guide

Targeted platforms

All Linux distributions (Suse, Mandriva, Redhat/Fedora, Debian/Ubuntu, Slackware, Gentoo etc..)

Requirements

  • [Mandatory] Java Runtime environment (JRE) 1.5 and above. 

    • If you are unsure of your Java version, open a console and type: java --version
    • If needed, you can get JRE for free at http://www.java.com
    • Caution: many distributions provide GCJ JRE by default: this will not work with Jajuk !

  • [Strongly recommended] MPlayer 1.0 pre8 and above available from the $PATH (this will provide better sound quality and more audio formats such flac)

  • [Recommended]  Mozilla 1.6 and above available from the $PATH (this will provide the Wikipedia view)

Installing JRE under Linux

Download the JRE RPM version (.rpm.bin file).

  • Lauch installer : sh j2re-{version}.rpm.bin

  • Log as root and install the RPM : rpm -ivh j2re-{version}.rpm

  • If needed, add the JRE/bin directory in your PATH (Sun JRE RPM is installed in /opt/java directory that is not in the PATH by default).

    • Add the line export PATH=/opt/java/jre1.5.0/bin:$PATH (change this by your own directory) in /etc/profile or ~/.bashrc for instance

  • Check you now use the new JRE by opening  a console and typing: java --version

Download

Download Jajuk last distribution at http://sourceforge.net/project/showfiles.php?group_id=91412. Choose one of RPM files.

Jajuk Installation

We provide two packages for Linux : Suse RPM (jajuk-${version}-suse.noarch.rpm) and Mandrake RPM (jajuk-${version}-1mdk.noarch.rpm). Suse and Mandrake RPM should work with most others Linux  distributions but menu icons may not be created. To install, use your favorite graphical tool (Yast, Drake...) or log as root and launch rpm -ivh jajuk-${version}.rpm. 

Notes:

  • It can take a while for shortcut to appear in menu.

  • Under Suse, shortcut is created in "Multimedia" menu section.

  • Under Mandrake, shortcut is created in "Multimedia/Sound" menu section.

  • Note that you can also install Jajuk under Linux using the Java installer (see main installation guides).

Launching

  • Use the new shortcuts in Applications/Audio under Suse and Multimedia/Sound under Mandrake

  • Or simply type jajuk in a console

Upgrade

To upgrade Jajuk, log as root and launch rpm -Uvh jajuk-${version}-suse.rpm

  • It is safe to install the new release over previous one without uninstalling, Jajuk supports backwards compatibility. Your collection and configuration is kept and is stored under {your home directory}/.jajuk

  • Jajuk doesn't support downwards compatibility: A Jajuk release may not work with a collection built by a newer release than the one you use. 

Uninstallation

  • If you installed using RPM, use your favorite graphical tool (Yast, Drake...) or log as root and launch: rpm -e jajuk-{version}

  • Remove {user home}/.jajuk directory for each user you want to uninstall
