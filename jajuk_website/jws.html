Launch jajuk online

Click on "Web Start me now" button to launch full Jajuk release without install. Current release is: 1.0.3
Java Web Start auto-detection :

Notes:
  • You need JRE 1.5.0+ and JRE 1.5.0_6 under Linux.
  • Jajuk is self-signed so you will be prompt to accept Jajuk launching: accept safely temportary or permently, it is exactly the same code than the standalone installer.
  • Application works offline.
  • [Linux only] Some JRE web start engine are buggy (nothing happens, 100%CPU), JRE 1.5.0_06 + should work.
  • [Linux KDE only]  Automatic shortcut and menu creation doesn't work, install RPM instead or create shortcuts manually with following command: /{JRE path}/bin/javaws http://jajuk.sourceforge.net/jws/jajuk.jnlp or launch javaws, select jajuk and execute.
  • [MacOS only] Upgrade from 1.4.2 to 1.5 is not automatic, you have to select 1.5 in the 1.5 configuration panel, make sure that 1.5 is the first in the JVM list.
